其实我还健在

发表于 2007-11-16 | 更新于 2018-05-24 | 分类于 生活

这篇博文没有主题，只是告诉各位光临的过客，我至今仍然健在。我也想为这段时间的无作为举点理由，但嫦娥登月我错过了登机、华南虎那会我相机没电、拍《色戒》赶上我没档期、制定新假期制度时我又在加班，于是……

还是汇报一下工作吧：

加班天昏地暗

最近公事N多，升级/故障/需求/中测，压得喘不过气来，频繁地通宵。整得白天瞌睡眠眠，夜晚精神亢亢。好不容易躺下，还做梦梦见电话在响，惊醒后发现电话果然在想。。。额的上帝老天爷啊！

消费日日高攀

以前读书时穷惯了，省吃俭用，但也积累了不少数码产品。现在经济终于宽松了点，除了买了个相机看得摸得着外，不少Money都入了肚肠。一边在宿舍交足了伙食费，一边三天两头的在外吃饭，不是我请同事就是同事请我，出门抬手就拦车，付钱时还假惺惺要张发票。

基金石沉大海

平日里不少同事在玩基金股票，耳濡目染受多了影响，又因为是牛市，也有些跃跃欲试。 10.11号正好广发聚丰开放申购，试着买了1000块，没想到卖得太火。最终配额四份之一，于是我的第一份基金成了250，当时大盘是6000多点。后来又入了几K其它两种基金，结果连续赶上几波大跌，前两天竟然探到5100左右。NND，时不待我啊，基金竟然都能被套牢。还是眼不见为净，尽量不看，心中却日日盼涨。

心中拔凉拔凉

生活这么没劲，心里自然爽不到哪里去了。最近也没出什么好电影好小说让人以寄托，每天就是打开抓虾欣赏下各位牛人的博客，找点冷笑话自讨开心而已。现在就盼着中测能顺利度过，还我以正常生活。或者时间有多还能拾起编程旧事，写点有意思的东西。

料理鼠王(Ratatouille)

发表于 2007-10-20 | 更新于 2018-05-26 | 分类于 扯淡

我一直认为，皮克斯之于动画就像暴雪之于游戏——一旦出品，必属精品！

昨天晚上挂了一通宵，终于下到了皮克斯制作的新片《鼠王料理》。不知道是最近关心太少还是迪斯尼没花大力宣传，我事先竟然不知道一点有关这部片子的消息，还沉浸最近的《汽车总动员》之中没缓过神来。

故事发生在美食之都巴黎，讲的是一只非常有烹饪天赋的老鼠和一个非常没有天赋的厨师，老鼠藏在厨师的帽子里，提着他的头发像操作提线木偶一样帮厨师烹饪，最终成就了最无可挑剔的美食。

看完了这部片子后，没有期待中非常过瘾的感觉。我尽力让自己以为这是我胃口太刁所致，不愿意承认皮克斯终于也有力不从心的时候。《鼠》画面上不如《超人总动员》和《汽车总动员》绚烂，剧情感觉老套，创意也没太多吸引人的地方（还记得《蚂蚁总动员》中那只小蚂蚁用露水和叶子造的望远镜么？）。

此片最让我印象深刻的地方是一首法语插曲——《le festin》，传上来分享。

办公室下移

发表于 2007-10-11 | 更新于 2018-05-25 | 分类于 工作

因为某些原因，我们的神州行反复下移推迟，但今天我们的办公室给下移了——从原来网管大厦的第9层搬到了另一栋小楼的第4层。

该楼在网管大厦的后面，一共才4层高，而且没了电梯，加再自己所住的地方是6楼，现在每天要爬20个楼层了。这原来省呼叫中心工作的地方，想当年估计美女如云，但现在她们都搬到金源大厦那边去了，曾经有幸去她们的食堂吃过一次饭，被MM包围的感觉颇为飘然。这的办公条件要比原来差不少，原来每个人有一个不小的隔间，现在则是那种网吧一样的桌子，大家都坐下后就像在学校机房上课；天花板也矮了些，多了些压抑感。屋顶的两个角里有分别有一个全景摄像头，应该是以前装上的，不知道以后会不会有人天天盯着我们。还有，怎么空调都还没装？

还是Google好

发表于 2007-10-10 | 更新于 2018-05-24 | 分类于 扯淡

前不久写了篇《还是家好》的博文，今天闲着无聊用Google搜索了一下，发现在搜索结果中排第二，再用Baidu试试，在结果前10页都没有发现。观察到一个现象：Google的搜索结果中第一页几乎全部包含“还是家好”这个词，从第二页末开始虽然拆成了“还是”和“家好”两个词，但仍然出现在同一个结果里；而Baidu在第一页的结果中就已经只剩下“家好”了，而且前几名全是些公司的链接，极度怀疑是出钱买到的排名。

再做了两次搜索比较：

关键字——“cpiz”：
Google的搜索结果中本站域名排第一位；Baidu的搜索结果非常令人失望，在前四页中都没有找到直接链接，唯一一个靠点谱的还是我挂在Feedsky的Feed，而且排在第四位，第一名是奇虎论坛搜索，Baidu难道要靠奇虎支撑自己的搜索了么？
PS：百度的结果中还能找到我以前在MSN Space文章，而我早已经删除了那个空间，可以想象：用百度来学习历史应该会事半功倍。

关键字——“site:cpiz.com”：
这是站内搜索，Google搜索出了553个结果，Baidu有247个结果。但Baidu的结果都是过期的，我的Blog标题早换成Cpiz’s Blog了，而百度的结果仍然是Thinking Life。

这几轮比下来，谁优谁劣立见分晓。这里要提到一个前提，本站是使用了Google Sitemap插件即时通知Google收录的，对Google的搜索有优势，但百度没有类似的WP插件也是一项它不如Google的地方。

我以前非常喜欢Baidu，认为它在中文搜索上要强于Google（忘了是怎么得出这个结论的了），而且Baidu有一个“我知道你不知道我知道你不知道”，“我知道你不知道我知道你不知道我知道你不知道”的广告非常有印象，但在事实面前，不如就是不如。从民族情节上我还是要支持Baidu，希望它能自强不息，多给国人些惊喜。

还是家好

发表于 2007-10-06 | 更新于 2018-05-26 | 分类于 思考

10月3号晚上，漫长的三个半小时车程之后，我回到了萍乡。以前一直生活在这，没觉得有什么变化，上大学后几个月才回来一次，终于承认每一次我所见的萍乡都是新的。

老妈仍旧是在门口等着，Taxi一停下就迎了上来，比上次见她又胖了点，我没说，知道她不喜欢。见家门后看到了外公外婆，仍然神采亦亦。表弟的变化比萍乡快，现在已经是个小伙子了，仍旧不太爱说话。他的妈妈，也就是我舅妈，还是那副笑盈盈的样子，顶着大肚子。据说预产期在7号，我希望这次能多个妹妹。打遍招呼后开始吃饭，虽然辣椒比我希望的少，毛豆比我预料的硬，但给我的感觉还是塞过吃酒店。吃这一点我还是比较容易满足的，但又最难满足，因为除了萍乡，其它地方做菜都不怎么放辣椒。

第二天陪妈去逛了下街，给家里餐桌置了块漂亮的桌布。她说最近心绪不好，我们到药店挑了瓶药，140块，趁她掏钱时我抢先付了，营业员羡慕的眼神下，我知道妈很欣慰。工作半年了，能用钱买到爸妈的笑容，累点也值。

晚饭上终于吃到了足够份量的辣椒菜，大呼过瘾，掏出相机拍了三张，现在看看还口水直流。有些东西，真的是无法改变，也不愿改变。



两天的生活很简单：看电视剧《士兵突击》和《丑女贝蒂》、吃饭、睡觉。今天大表弟来了一趟，这两年他一直在外面打工，都没见着面；如今回到了萍乡工作，还诓了个MM回来，我妈借题没老少唠叨我找女朋友的事。

好日子总是过得很快，明天又要回南昌了，没意外的话，下一次回来就是过年了。

国庆加班日

发表于 2007-10-02 | 更新于 2018-05-24 | 分类于 工作

自打接手帐管的工作起就知道以后五一国庆是都不用想放假了，对这次国庆的加班，我一点也不吃惊。我只希望可以在123号里把事情赶快完成，可以早点回家休息几天。

进公司时还是昨天13:30，今天出来时已经是08:20了 ，该死的积分和冲销程序，不停地出问题，让我一夜没合眼，靠咖啡顶着才没睡着。

回到家刷下牙洗把脸 ，刚睡下一会，就被电话叫醒，据说全省缴不了费了。吓得我立马跑回去救火，解决问题后再次走出公司已经是12:50，差点就打破连续24小时不眠工作记录。

下午睡了一会，到六点钟的时候又睡不着了，老惦记着积分的事情，吃了个泡面，洗了个澡就过来了。谢天谢地，目前程序运行正常，希望明天可能回家~！

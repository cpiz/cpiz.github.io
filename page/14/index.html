植树节

老天又一次欺骗了我,满以为春天真的来了,谁知今天又大风降温,就差没下雪了.听新闻讲,这次天气变化是全国性的,今天可是植树节,冻死了花花草草就不好了啊.

上一次种树还是小学时学校组织的,那时候嘻嘻哈哈,把树苗当草玩,造成的结果就是大部分树苗还没长到草高就死去,剩下的几棵孤伶伶特显寒碜.大了后常常吃完晚饭陪妈妈到学校操场散步,看到这情景很觉得遗憾.

今天就来个画饼充饥,在这种棵虚拟的吧.

春

发表于 2006-03-03 | 更新于 2018-05-24 | 分类于 生活

啦……啦
哟 哟 哟 哟
今天天气不错
挺风和日丽的
我们下午没有课
这的确挺爽的
………………

昨天睡得太晚，一早起来发现一脸的豆子，提醒我原来我还依然青春。顶着瞌睡和烦燥，上完两节软件体系结构，走出教学楼，原来天气如此之好。明明前两天还寒冷霸道，冻得我手脚麻木，希望这次的太阳是货真价实的春日。

回到宿舍，像往常一样开机。在明朗的光线下，键盘的污浊更加触目，不知道的人也许会以为我平常是用脚来使它。拆开来放水龙头下冲，放洗洁净洗，用牙刷刷，然后挂衣架上晾。还给它拍了张照片，这种情景应该不常见到吧，不过后来中午去食堂的路上看到有人有扁担挑了台电脑，跟他比起来我这就不算什么了。

开学到现在已经过了两周多了，境遇和这两周的天气一样，由好越转越坏。奖学金终究是没能拿到特等，重修也还剩下两门，另一路上也是荆棘漫漫。转眼也是要毕业的人了，回忆回忆还是有些遗憾。

电子商务时代

发表于 2006-02-18 | 更新于 2018-05-24 | 分类于 生活

电子商务这个词N年前就吵得沸沸扬扬,但我一直觉得和我关系不大,但这几天着实体验了一把.

事情还得从上学期买手机的事说起,那会儿差点就从淘宝上买了个SPV E200.最终还是找个个南昌卖家本地交易,那天还借了harry他400块钱.这开学来办的第一件正事就是去开通了网上银行业务,因为他现在去了上海,从网上还钱给他比较方便.有事没事登录银行,看着money乖乖的显示在屏幕上,很有一种守财奴的满足感.给他还钱时使用的是传说中的支付宝,从银行里转450块到我的支付宝帐号上,然后用支付宝给他即时汇款,几分钟就搞定,也没花一分钱手续费,实在是方便.

支付宝里还剩下50块钱,早就想好了怎么花,上淘宝买了个心仪已久的LED手电.东西只花了21块,邮费却还要用掉10元,有点心疼.没事的时候在淘宝逛逛,发现好东西真是太多,以

前那些只想过却从来没在商店里看过的东西都能找到,运气好的话还能通过拍卖的方式买到非常超值的东东.昨天晚上趁着电骡在拖red hat 9在淘宝上闲逛,今天忍不住买了个调频转发器,数字万用表和一个12倍单筒望远镜.可惜的是本来只要花16块就能拍到了一套26合1的工具组,因为出价早了4分钟,被人在最后一分钟内出价超过,没有买成.

两天之内我已经在购物上花了一百多块,却连钞票都没有摸过,这种用鼠标花钱的方式实在是刺激.现在上网闲逛看到弹出的淘宝广告似乎也不那么讨厌了:)

在家半月

发表于 2006-01-20 | 更新于 2018-05-24 | 分类于 生活

不觉间已经在家呆了半个月了，开始想念学校的生活。唉，人总是这样不知满足。

这半个月每天最主要的活动就是睡觉看小说还有吃，有的时候还是睡在床上边看小说边吃零食，毕竟这样时间利用率最高。家是物质生活的天堂，光零食就六七种，相比在学校想吃舍不得买的日子只能用奢侈来形容。

有事没事就和老妈去看看新家，想象它装修完毕后的样子。老妈惦记着新房子好多年了，现在终于如了愿，又急着想早点装修，巴不得年前就搬进去。到时候我的书房一定很爽吧，终于要有个书柜了。

考试成绩终于出来了，不仅心中大石落地，还得了个惊喜，成绩都是奇迹般的好，奖学金是没得跑了。以前听过种说法:大学生之所以为大学生，因为可以把一个学期的课程在考前的一个月内学完。如此我便可以说是大学生的典范了。要是能重修完n年前欠下的那三门数学那就完美了。

很久前就少鞋裤穿了，一回来老妈就说要带我上街买，我怕麻烦一直就拖着。今天耐不住她磨，又正好要买些吃的过年，就出去了一趟。

上大学后都是几个月才回一次家，所以更能看出萍乡的变化。这几年来浙赣铁路电气化改造终于见了成效，一部分路段已经在通车了。城内新建了不少高大的建筑，又修起了步行街。安源开发区内的320国道全部加阔，还竖起了一路的灯，路上跑的16路几天前全部换上了新客车。所以谁要是说中国近几年没发展我肯定第一个站出来和他拼命。

因为我怕逛街麻烦，我妈也怕我麻烦她，所以一致决定在心连心把所有东西都搞定。几下就买了两条裤子，但找不到中意的鞋，然后去超市区买其它东西。我向来不喜欢逛商场，其实是怕碰上喜欢的东西买不起，光看的话又对自己太残酷。显然和我妈出来买食品不用有这种担心，因为她和我一样馋，我们口味又差不多，而掏腰包的也是她。买到最后我都心疼她钱，把她拉了出来。最终这些吃的绝大部分都得落在我的肚里，希望别再胖了才好。

回家的第一天

发表于 2006-01-07 | 更新于 2018-05-24 | 分类于 生活

摆脱了烦恼的考试，告别了瑟瑟的寒风，终于回到了温馨的家。

说实话，想回家已经不是一天两天了。倒不是因为离家有多久，实在是被考试的压力和食堂的伙食所逼。平常逍遥惯了，到要考试时才急得茶饭不思，加上天气转寒和恶劣的食堂伙食，一天我也不想再等，昨天考完最后一门今天就走人。

昨天和朋友聊手机到一点多，两点才真正开始睡觉。为了不错过火车，还特意把SIM装再了N3108上，就因为它的闹钟功能最可靠。闹钟时间定在了6:15，结果早上我在闹钟叫响的几分钟之前就醒了，不禁感叹意识加生物钟的效果真是不可思议。

冬天早晨的这个时候天还完全是黑的，本以为公交车上回没几个人，事实证明我错了，幸好我找到个座位。汽车行驶的时我看着窗外想了堆乱七八糟的事。无论是火车汽车，我都爱看车窗，从火车的窗能很容易看到各地的风土，从汽车的窗能很容易看到人间百态，而我只须坐着。街道渐渐拥挤起来，行人也越来越多。我观察猜测着他们奔波的目的，上班？上学？锻炼？我们最大的区别应该是他们早起离家，而我是为了回家而早起，我大概是最幸福的吧。下了车到火车站还得步行几百米，看到一位妇女在叫卖报纸，一块钱三份。心头一阵发酸，她今天肯定起得比我还早吧？一年四季雨雾寒暑，生活真不容易，或许几年前她还是令人羡慕的国企员工吧？我伸手摸向口袋捏了个硬币，又松了开来。这短短的一两百米路，前面还有n个这样的妇女，生活得都不容易，给哪个都不公平，我也根本不需要报纸。

顺利地上了火车，应该是大部分学校都还没考试完，所以没多少学生，一点也不拥挤。睡了会觉玩了会手机聊了会天四个小时就过去了，火车难得地没有晚点。顺利地下了火车转上了公交，12:45进了家门。

和爸妈高兴地说了几句，然后开饭！菜都是我最爱吃的，夹上一块香辣排骨，那久违的味道让我感动几乎流泪，即使是块豆腐也比学校的鸡腿要好。下午补了个觉，一头倒在我软软的大床上那霎又感动了一次。终于苦尽甘来啊，四九年国庆的时老百姓的喜悦估计也就这样吧。

吃过晚饭陪妈妈散步，听她发这几个月积累的牢骚，回到家里吃零食看电视。家就是好啊，冷了有电暖器，饿了有方便面，渴了冰箱里有牛奶，馋了有大堆零食，想洗澡有热水器，学校里一样都没。

电视看得无聊了爬到床上用手机上QQ，然后开始写这天的故事。回家的第一天就这样过去，明天来制定个假期计划，因为开始想念我的电脑了……

于2005末

发表于 2005-12-29 | 更新于 2018-05-24 | 分类于 生活

年初的时候写票据总是会习惯性地在日期那填个2004，到月份时才想起该写2005了，然后把它揉成一团，再拿张新的……现在总算习惯了2005，却该换成2006了。

2005年1月
在Joyes发布Nokia小D电子书生成器

2005年4月
初学C#，完成大学生挑战杯项目，三等奖。
相同项目提交学校团委，参加“五四”青年杯电脑作品竞赛，软件设计类作品一等奖。

2005年6月
参加CET4考试，奇迹般过线。

2005年6-7月
期末考试，分数竟然够得奖学金。

2005年7-8月
实习，学习ASP.NET，独立完成公司一个小项目。

2005年9月
大三第一学期开学，学习java游戏开发，j2me，顺便学了下ASP。

2005年9-10月
用j2me开发Nokia手机电子书阅读器，技术完全自主。

2005年10月12日
软件在Joyes发布，同日“神舟6号“发射成功（不是我发射的）。

2005年11月
退出bbs.ncusc.com

2005年12月
参加离散数学、数字逻辑、高等数学重修考试。相继失败，06再来。
总得来说还是收获颇丰的，学习了.NET和java，过了四级，里最充实的一年。
05年是01级师兄师姐毕业的一年，也是02级朋友准备毕业的一年，对自己的将来也多了几分担心，伤感的一年。
05年末身上还背了900块的债，同时又400块借债在外，最穷的一年。

================故事的分隔线===============
前天，参加java考试，应该考得不错。
昨天，得知网络考试卷子改出来了，今天给老师发了短信，82分，意味着那天的舞弊风波平息。
今天，发下了200块奖学金。终于可以不用再借钱，又可以回到惬意喝可乐的日子。

初恋50次

发表于 2005-12-27 | 更新于 2018-05-23 | 分类于 生活

好的电影能从屏幕直通心灵，触动你最敏感的那根神经。最能发挥这种效果的影片，我印象里非好莱坞荒诞剧莫属了。富有想象力的美国电影人，创作出那些不可思议的故事，让心灵直面最深的考验，也赢得了票房的丰收。其是令我印象颇深的有《土拨鼠日》、《男人百分百》等等，而华语影片却一部都数不出。

今天的这部《初恋50次》，令我印象深刻。不用演员多出名，场景多华丽，仅这个剧本就足够人心里暖一阵的了。

亨利是夏威夷水族馆的兽医，因为大学里初恋受过伤害，从此对爱情的长久不再相信。利用夏威夷旅游盛地的特点，装成各种职业与前来渡假的女人们玩着不用负责任的爱情游戏，当着他快乐的单身汉。亨利也有着不凡的梦想，并一直在为之努力，那就是攒钱买艘船到太平洋里去研究海象。

一次偶然的机会让他在一家餐馆里见到了那个不能让他忘怀的姑娘，第二天早早的来到这里等着她的到来。刻意地制造了机会后得到了和姑娘同桌吃饭的机会，姑娘叫露西，一番交谈后彼此都有了好感，并约定第二天老地方见面，亨利的心中重新燃起了爱情的火焰。

意想不到的是第二天亨利开心地和露西打招呼时，露西却像和从没有见过他一样，觉得亨利搔扰了自己。郁闷不已的亨利从餐馆老板娘那里得知了真相：一年前的10月13日露西父亲的生日，父女俩按着惯例去采一个菠萝，在回家的途中发生了一场严重的车祸，露西因头部受伤住了三个月的院。因为伤到了脑内的记忆组织，她得上了一种罕见的失忆症，只能记得事故以前的事情，而之后的每一天她都没有印象，就像永远活在了那一天。露西的父亲和哥哥为了不伤露西的心，每天也都装作是在过去年的10月13号，给露西准备那一天的报纸，用录像带播那天的球赛，甚至每天把房间墙壁刷成白色来给露西第二天进行创作，露西身边的朋友也都在善意地为她掩瞒着真相。老板娘告戒亨利不要再接近露西，但陷入情网的亨利不能自拨，想尽了办法每天都餐馆去“认识“露西，这行为触怒了露西的父亲，严厉禁止亨利再去餐馆找露西。亨利无便把邂诟的地点改在了露西必经的车道上，制造出汽车抛锚、道路失踪、糟人抢劫、被人绑架等各种事件让露西停下车来和他认识，甚至还请来了水族馆可爱的企鹅来拦道，令人喷饭。认识亨利的日子让露西获得了久违的快乐，亨利的真心也打动了露西父亲，于是不再阻拦亨利的行动。

完美的计划也有失算的时候，偶尔某天露西还是能发现事情的真相，于是大家一次次带着露西去治疗她的医院证实，不过第二天露西又会重新忘了一切。亨得不再愿意露西天天活在他们的安排中，尝试着在露西每天醒来的时候就用录像带告诉露西真相，于是露西又可以开始新鲜的生活。不用再费心思花在“认识”上让二人有了更多的时间来享受爱情，露西也日日能得到甜蜜的初吻。亨利越来越爱露西，甚至决定了为她放弃自己航行的梦想。

每日重新爱情的爱情终于发展到了发生关系的程度，可当第二天露西睁开眼睛发现光着身子的亨利时，歇斯底里地扁了亨利一顿。重新看过录像带后知道了真相的露西开始思考这样的生活方式是否合适，她觉得自己影响了亨利的生活，希望亨利别再来找她。

后来的日子露西主动地申请去医院给同样有失忆症的病人们教画画，亨利忍着思念不再去见她，决定开始自己航行。开着船离露西所在的小岛越来越远，听着露西唱着的歌，亨利终还是忍不住对露西思念，掉转船头寻找爱情。

曾经多日的相处还是给露西留下了朦胧而又深刻的记忆，二人最终走到了一起，露西开始迎来每日重新认识一次女儿的日子……

荒诞的故事，幽默的情节，温暖的感情再上众位演员出色的表现，让我在一种快乐和温暖中氛围中度过了100分钟，也留下了思考。假如可以选择失忆，你想忘却什么？能保证以后不会后悔吗？假如某天爱上一个失记忆的他/她，你会愿意每天和她/他重新开始吗？多久你会厌烦？这些我不敢想像的问题，真希望我知道答案的时候还能够欣慰，或许一辈子也不会知道答案。

和暂时失忆有关的爱情片这并不是第一部，早两年前香港就有一部《我的失忆男友》就和它非常想像，但最终能留在我心上的还是这部《初恋50次》。找准人们最需要的感情，配上高水平的演员，低成本也能创造奇迹。好莱坞虽然商业气息浓重，但中国电影还有很多的地方要学习的地方。

郁闷的网络考试

发表于 2005-12-23 | 更新于 2018-05-23 | 分类于 生活

今天上午9点考计算机网络，因为之前要准备重修和前天的算法设计，昨天才开始有时间来准备。昨天上午快9点了起床，去洗个澡洗把衣服，把手冻得冰凉。去管理员那烤烤电暖器，结果坐那聊了一上午天，下午1点才去自习。

一直以来都是很喜欢网络的，但急功近利，喜欢读那些实际的知识，对其基本原理却没有花心思去了解。这学期开了网络课，又因为种种原因，翘了太多，去了也是根本没听，听了也跟不上。还好基础和兴趣帮了大忙，加上教材非常不错，昨天自习时越看越有劲，以前很多模糊和不了解的问题一个一个清晰起来。昨天看到凌晨两点，加上今天早上的半个小时，终于把这本书要考的部分读完了。

兴趣和动力相加，效果就是不一样，一学期的网络课程我12个小时就学完了，但我信心实足，相信绝对不比那些天天听讲的同学理解得浅。考试很顺手，及格是肯定没问题了。有些题目还要再斟酌一下，我没有立即写答案，跳过去做后面的。做完最后一题，舒了口气，准备跳回去解决前面留下的。这时前面同学问了我一个问题，我半天没听听他说什么，回了一句“听不清”，结果双双被老师没收了卷了。

郁闷啊~

平淡就是幸福

发表于 2005-12-18 | 更新于 2018-05-23 | 分类于 技术

前不久买了个智能手机，这段时间天天把玩着，爱不释手。昨天它突然出了毛病，死也不认MMC卡。虽然手机自身带了32M的内存，也不算小，但这意味着以后再也不能看电影和听音乐。950大洋买的高级玩具突然缩水大半，想想我就心痛。
疯狂地在各大论坛找资料查原因，最后从一专业维修人士那得知可能是键盘板坏了，帮我换一块得200RMB，但是他也缺货。正自郁闷之时，同学揣着个邮包闯进寝室，来找我借读卡器。他订的SPV刚刚到货，带了张128M的SD卡，现在要借读卡器去装软件，于是我心痛变成了心酸。
化种种刺激为动力，凭我资深DIYER的精神，死马当活马自己修一回试试。这手机去年4月上市的时候，行货要卖近5KRMB，绝对算是一精密仪器，动手拆它，需要勇气。拔下电池，取出SIM卡，打开后盖，取出主件，取下二个螺丝，小心取下键盘板，要找的SD卡槽就装在上面了。小小的一片就要卖200块，科技果然是第一生产力……拿到阳光下检查一番，可以看到上面的电路和元件做工非常精细，也算对得起它的身价了。发现卡槽里有一片小小的纤维絮，肯定是放口袋里钻进去的衣服纤维，用镊子小心夹去，拿毛刷对各处清理了一番，再仔细地检查了SD卡槽的触片。最后忐忑不安地装机，开机，启动软件检查。或许是上天照顾，或许我的确太强-_-!!!，久违的Storage Card字样重新显示在了LCD上，那一刻，真是无比开心。
最爱的是得不到的，最痛的是刚刚失去的，想必人人对此都有体会，所以星爷那段“曾经”的台词才能如此地广为流传。人们最持久的是占有欲，最短暂的是满足感，所以都说自己不幸福。其实幸福很简单，是久别归家后父母的笑脸，是饥肠漉漉时泡上碗一块五的方便面，甚至是逛街逛到内急，抬头发现远方隐约“公厕”二字。
平淡中处处皆是幸福。
这想法我很早就有了，只是从没今天般强烈。我知道我很快会忘却，即将再次投入各种麻烦的旋涡，但我更知道某天我会再次有这种感觉。只希望再也不会忘记它时，我依然年轻。

12月16日情

发表于 2005-12-16 | 更新于 2018-05-23 | 分类于 生活

12月16，对世界来说是个普通的日子，没有节日，没有大事件，每天依然那么多人出生，依然那么多人死亡。对于我，不一样。这一天结束了很多事，又开始了不少。

12月16日的凌晨两点多我才开始睡眠，早晨依然7：40起床，去上web开发最后四节课，中午和老同学吃火锅，吃完遛了回街，回到学校直接上下午的java，得知只要交完了最后的大作业下周就可以不用去了，全班目前交了两个人，我是第一个。java的结束，标志着大三上学期的所有课程的结束，开始进入复习备考阶段。

java课上收到高中朋友的短信，要借500块钱，看语气是十万火急那种。心里很难受，风光的时候不曾联络，落难时总能想起我。朋友该是分享快乐的那种人还是救你水深火热的那种人？自己实在是拿不出来，最近为了买手机已经借了同学500，但连连收到他那苦苦请求的短信，还是不由得心软了，再厚一番脸，给尚有余钱的同学发短信……问他借钱的原因，他吱吱唔唔，估摸着是闯了什么祸事了。朋友，希望你不要欺骗朋友，希望下次发短信来时能让我为你高兴一回。

12月16日，是高数重修考试的日子，也是我三门重修的最后一门。卷子一发下来，傻了……浑浑噩噩涂了几把，交了卷走出了教室，碰上三个另一考场的同学，脸上忍不住的笑，直呼抄得痛快。又是一个人，又是清静的夜，又是那么明亮那么圆的月亮，又是从北到南的路。今年的重修结束了，明年的重修刚刚开始……

不要为过去哀怨，眼睛要看着前方。

“书法家”

发表于 2005-12-04 | 更新于 2018-05-23 | 分类于 生活

刚学会提笔写字的时候正值五年迫害时期，小学五年里那个班主任现在想想还颇有余悸。那时每天的作业都是抄N遍当天课文的词语解释，每个寒暑假的作业就是抄N遍本个学期课文的词语解释，一本小学语文词典抄了不下十回，熟练到对每个词的反应近乎条件反射，词语还没报完就能把解释背出来。背不出来的话，当然是要挨索子抽手心的，随机附赠的还有拧耳朵、抽耳光、自残（就是自己拿自己的手去拍桌沿）等等等等，反正是五花八门，相同点就是每种方法都能让你痛不欲生而又不会留下生理上的后遗。

要相信人的适应能力是很强的，无论多么艰苦的环境都能找到苦中作乐方法，更何况是纯真的童年。那时娱乐项目之一是作业比赛，老师常常把作业布置下来就不知闪到何方去了，我们便在教室里比赛，一时整个教室里都是沙沙的写字声，然后某人把手一举，高呼一声：”我写完了“，引来数十双羡慕的眼神（羡慕他这么快就脱离了苦海），了解到差距后各自计算一下成为第二的可能性，继续奋斗。班上有多少人语文出色不敢说，反正我们就是不缺书法家，还都是狂草派的。那个年代还培养出不少走偏锋的人才，能一手握两支甚至三支笔写字，速度自然不用说。

还记得那时写得最快的“书法家”的家伙叫夏勇，常常能得到老师的“关怀”，一顿打骂自不用说，时不时还给些锻炼的机会——重做。没想竟然逼出了那个家伙对写字的兴趣，还报了了书法班，有空就练练字，当真造出一个书法家来。我当时也算半个“书法家”吧，偶尔能得个第一，但老师的“关怀”就少了很多，最终没能成为真正的书法家。

老师还不急，我爸先急了，暑假时要我在家练正楷。我一笔一画的写起来那字倒也还是个方的，但真正的要写字时就变得不伦不类了，至今如此。

说了那么多历史，其实今天要说的就是：别人介绍给我一个软件，可写出漂亮的毛笔字，叫Ougishi。唯一的缺点是它竟然是日本人编写的，实在是国人的悲哀。打开软件，在画板上小心地写个汉字，就可以调整它的草书、笔脉、轻笔、连绵、宽度、抑扬、飞白，然后点击“变换”，刚才可以说丑的字立马就变得有书法的味道。还它预设了行书、草书、王羲之、怀素、徽宗、苏轼等十来种风格。

CET

发表于 2005-11-30 | 更新于 2018-05-23 | 分类于 生活

最近考试临前，急得我焦头烂额，常常感叹：我这大学前两年是不是太背了？周围的有爱情丰收的、钞票丰收的、学业丰收的、仕途丰收的。而我爱情没找到，奖学金没见到重修费倒交了近千，考试挂了三科，官没当上半个。也许唯一让我有点欣慰的就是第二次四级考试过了，可能过了。

英语是个不错的工具，但我不喜欢别人逼着我学。中国教英语这种一个一个语法、一个一个短语细细分解的方法，我实在是接受不了，听了多少遍就是记不住。一直以来考英语靠的就是感觉，初中英语简单，靠感觉能够应付，但到高三时也只能把成绩维持在及格线上下了。记得高考英语似乎是及了格，也可能是没及格，反正如果没有瞄到边上同学几个题的话那就是一定没及格。

还没上几天大学就听到学长们教导，无论什么专业，英语都是最主要的学科，只要四级过了大学就相当于完成了一大半。我觉得还不用急，总想着四级还远着，等到了再说。英语老师好得那叫一个菩萨，考试时范围一划稍稍一背就能pass，于是我也从没认真看过两眼英语，好早买了一本崭新的四级词典现在仍然是崭新的。

就这样混着过了一年，大二上学期第一次四级考试终于来到，我仍然没怎么急，反正看看运气试试水平。只记得考试前一天晚上QQ上的老同学从没在线得这么齐，大家头像一齐闪动，那叫一个壮观，都是在互相打听和传播“内幕消息”的。我那天听到N个版本的据说是第二天考试的作文题，找了N个英语好的朋友来帮我写作文，我只管背，又听到据说几点几点谁谁谁能弄到答案。反正那天晚上是一点没睡好，第二天早上7点就被人电话吵醒，还是离不了这事。

考前十几分钟竟然肚子疼，去上了趟厕所，手机收到好心姐姐发来的答案，马上回到考场，趁老师检查准考证的把答案抄到了桌子上。卷子一发下来就傻了，答案与选择题的数目都没对上，作文也没押中，叹了一声，只好DIY了。放听力时太紧张，一走神就漏了几个，瞟了边上同学几个答案。听力完后先做阅读，做了两篇觉得好难，跳过去去做完形，觉得完形也很难，又去做单选，最后发现单选比阅读和完形都难。权衡一番后还是先做好完形，然后回到阅读。当老师说15分钟后收第一卷时我就傻了，阅读还剩一篇，单选还有几十个没做。这点时间只够我填答题卡了，硬着头皮把卡片填满，交卷。开始作文，咦，四级考试怎么才留着这么小一块地方写作文啊？不管它，我写满它也就能凑个辛苦分了吧。写完作文还剩十五分钟，我打开收音机，耳机里传来声音“魅力934……”

时间一到，考试要结束了，看见别人在收拾卷子，咦，怎么这边也有格子？才知道还有16开的一面可以用来写作文，就我刚才写的那么一点，想拿辛苦分还差得远呢，倒扣分还差不多……

考完出来也没觉得很失败，毕竟自己平时没有努力。只想分数快点出来，看看我还差多少，估计也就是个30来分。真正分数出来的时候，我倒还乐了一下，48，离及格也就差12分嘛，是很有希望能捞到的。但后来看到一些朋友拿到了四级证，心里还是有些酸酸的。

没想到到了大二下学期就没英语课了，我想认真听一听英语的计划泡了汤，加上忙一些事情，第二次四级考试很快到来，我单词仍然只看到abandon，我现在能打出这个词来还是刚刚翻了一下字典。

第二次考试情形简直是第一次的翻版，仍然有几十个选择没时间做，我连转笔都用上了，好歹把答案填了个满。因为第二卷还多了道不知道叫什么名的题，作文都没时间写，仍旧是短短的。

几个月后查分，430，一个月后得到分数线，425，折算成百分制连61分都不到。周围第二次考四级的同学没两个上了这线，我高兴一番后开始怀疑，我是不是查错了，空欢喜一场。

今天，发下了四级成绩单，颜色都和没过线的不一样，终于踏实了。最大的感觉就是：这四级也太儿戏了吧，同样一个人，同样的水平，前后两次考试的水平怎么就相距这么大捏？

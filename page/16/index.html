SPV E200入手

发表于 2005-12-08 | 更新于 2018-05-24 | 分类于 生活

受不了Windows Mobile的诱惑，不顾钱包空空，昨天入手了一台SPV E200。机子成色还不错，950块带了二电一充和数据线，就是有被非专业拆过的痕迹，左右软键和五向键的向下有点不太灵活。为了挖掘智能手机的功能，还到新大地买了读卡器/30元，蓝牙适配器/60元，耳机+转接头/40元，手机屏幕贴/3元，总共花去了1K1，这学期要借钱过日子了。

手机交易和购买配件从中午耗到下午五点，误了三节算法设计课了，希望老师考试重点别讲得太多。回到寝室开始玩手机，先恢复出厂设置，再重新装软件。windows mobile装软件的方法算是比较麻烦的，特别是装第一个软件，没有资源管理器，得通过同步软件在windows/start menu里建快捷方式才行，装好资源管理器才可以通过手机浏览文件。奇怪windows mobile都做到2003了竟然会没有自己的资源管理器，ie却照样捆绑。蓝牙的设置很麻烦，搞了半天才连上，同步软件里选择共享电脑的Internet连接，手机就可以通过电脑上网了，速度虽然没电脑快，但以后在被子里上网不成问题，试了一下上MSN，很不错，QQ没登录上，也许要换个软件。再试了VNC远程控制，全E文的软件，看得头大，还是没搞定，半夜一点多了，睡。

“书法家”

发表于 2005-12-04 | 更新于 2018-05-24 | 分类于 生活

刚学会提笔写字的时候正值五年迫害时期，小学五年里那个班主任现在想想还颇有余悸。那时每天的作业都是抄N遍当天课文的词语解释，每个寒暑假的作业就是抄N遍本个学期课文的词语解释，一本小学语文词典抄了不下十回，熟练到对每个词的反应近乎条件反射，词语还没报完就能把解释背出来。背不出来的话，当然是要挨索子抽手心的，随机附赠的还有拧耳朵、抽耳光、自残（就是自己拿自己的手去拍桌沿）等等等等，反正是五花八门，相同点就是每种方法都能让你痛不欲生而又不会留下生理上的后遗。

要相信人的适应能力是很强的，无论多么艰苦的环境都能找到苦中作乐方法，更何况是纯真的童年。那时娱乐项目之一是作业比赛，老师常常把作业布置下来就不知闪到何方去了，我们便在教室里比赛，一时整个教室里都是沙沙的写字声，然后某人把手一举，高呼一声：”我写完了“，引来数十双羡慕的眼神（羡慕他这么快就脱离了苦海），了解到差距后各自计算一下成为第二的可能性，继续奋斗。班上有多少人语文出色不敢说，反正我们就是不缺书法家，还都是狂草派的。那个年代还培养出不少走偏锋的人才，能一手握两支甚至三支笔写字，速度自然不用说。

还记得那时写得最快的“书法家”的家伙叫夏勇，常常能得到老师的“关怀”，一顿打骂自不用说，时不时还给些锻炼的机会——重做。没想竟然逼出了那个家伙对写字的兴趣，还报了了书法班，有空就练练字，当真造出一个书法家来。我当时也算半个“书法家”吧，偶尔能得个第一，但老师的“关怀”就少了很多，最终没能成为真正的书法家。

老师还不急，我爸先急了，暑假时要我在家练正楷。我一笔一画的写起来那字倒也还是个方的，但真正的要写字时就变得不伦不类了，至今如此。

说了那么多历史，其实今天要说的就是：别人介绍给我一个软件，可写出漂亮的毛笔字，叫Ougishi。唯一的缺点是它竟然是日本人编写的，实在是国人的悲哀。打开软件，在画板上小心地写个汉字，就可以调整它的草书、笔脉、轻笔、连绵、宽度、抑扬、飞白，然后点击“变换”，刚才可以说丑的字立马就变得有书法的味道。还它预设了行书、草书、王羲之、怀素、徽宗、苏轼等十来种风格。

CET

发表于 2005-11-30 | 更新于 2018-05-24 | 分类于 生活

最近考试临前，急得我焦头烂额，常常感叹：我这大学前两年是不是太背了？周围的有爱情丰收的、钞票丰收的、学业丰收的、仕途丰收的。而我爱情没找到，奖学金没见到重修费倒交了近千，考试挂了三科，官没当上半个。也许唯一让我有点欣慰的就是第二次四级考试过了，可能过了。

英语是个不错的工具，但我不喜欢别人逼着我学。中国教英语这种一个一个语法、一个一个短语细细分解的方法，我实在是接受不了，听了多少遍就是记不住。一直以来考英语靠的就是感觉，初中英语简单，靠感觉能够应付，但到高三时也只能把成绩维持在及格线上下了。记得高考英语似乎是及了格，也可能是没及格，反正如果没有瞄到边上同学几个题的话那就是一定没及格。

还没上几天大学就听到学长们教导，无论什么专业，英语都是最主要的学科，只要四级过了大学就相当于完成了一大半。我觉得还不用急，总想着四级还远着，等到了再说。英语老师好得那叫一个菩萨，考试时范围一划稍稍一背就能pass，于是我也从没认真看过两眼英语，好早买了一本崭新的四级词典现在仍然是崭新的。

就这样混着过了一年，大二上学期第一次四级考试终于来到，我仍然没怎么急，反正看看运气试试水平。只记得考试前一天晚上QQ上的老同学从没在线得这么齐，大家头像一齐闪动，那叫一个壮观，都是在互相打听和传播“内幕消息”的。我那天听到N个版本的据说是第二天考试的作文题，找了N个英语好的朋友来帮我写作文，我只管背，又听到据说几点几点谁谁谁能弄到答案。反正那天晚上是一点没睡好，第二天早上7点就被人电话吵醒，还是离不了这事。

考前十几分钟竟然肚子疼，去上了趟厕所，手机收到好心姐姐发来的答案，马上回到考场，趁老师检查准考证的把答案抄到了桌子上。卷子一发下来就傻了，答案与选择题的数目都没对上，作文也没押中，叹了一声，只好DIY了。放听力时太紧张，一走神就漏了几个，瞟了边上同学几个答案。听力完后先做阅读，做了两篇觉得好难，跳过去去做完形，觉得完形也很难，又去做单选，最后发现单选比阅读和完形都难。权衡一番后还是先做好完形，然后回到阅读。当老师说15分钟后收第一卷时我就傻了，阅读还剩一篇，单选还有几十个没做。这点时间只够我填答题卡了，硬着头皮把卡片填满，交卷。开始作文，咦，四级考试怎么才留着这么小一块地方写作文啊？不管它，我写满它也就能凑个辛苦分了吧。写完作文还剩十五分钟，我打开收音机，耳机里传来声音“魅力934……”

时间一到，考试要结束了，看见别人在收拾卷子，咦，怎么这边也有格子？才知道还有16开的一面可以用来写作文，就我刚才写的那么一点，想拿辛苦分还差得远呢，倒扣分还差不多……

考完出来也没觉得很失败，毕竟自己平时没有努力。只想分数快点出来，看看我还差多少，估计也就是个30来分。真正分数出来的时候，我倒还乐了一下，48，离及格也就差12分嘛，是很有希望能捞到的。但后来看到一些朋友拿到了四级证，心里还是有些酸酸的。

没想到到了大二下学期就没英语课了，我想认真听一听英语的计划泡了汤，加上忙一些事情，第二次四级考试很快到来，我单词仍然只看到abandon，我现在能打出这个词来还是刚刚翻了一下字典。

第二次考试情形简直是第一次的翻版，仍然有几十个选择没时间做，我连转笔都用上了，好歹把答案填了个满。因为第二卷还多了道不知道叫什么名的题，作文都没时间写，仍旧是短短的。

几个月后查分，430，一个月后得到分数线，425，折算成百分制连61分都不到。周围第二次考四级的同学没两个上了这线，我高兴一番后开始怀疑，我是不是查错了，空欢喜一场。

今天，发下了四级成绩单，颜色都和没过线的不一样，终于踏实了。最大的感觉就是：这四级也太儿戏了吧，同样一个人，同样的水平，前后两次考试的水平怎么就相距这么大捏？

从高等数学到《西游记》

发表于 2005-11-28 | 更新于 2018-05-24 | 分类于 扯淡

又要批评自己了，真不是推就不想动。放着N门数学重修不急，现在发神经又想做个俱乐部会员系统了。今天听到风声，前湖那边下周就高数考试，如果我们南区也同步的话，那不就是死期将至？无论是从形式上还是从良心上，今天都得去自习一次了。

难得来回自习就受回磨难，昨天才打了一天的球，落个腰酸背痛腿摔青，今天还是一楼一楼找教室直到顶楼才找到好位子。我已经不知道是第多少次翻开高数下册的第一章了，可每次的感觉都像是在翻一本从没看过的书一样。硬着头皮，看着那一团一团的豆芽符号，总算是背完了一张卷子（当然是最容易的第一张）。困意渐渐袭来，为了不着凉，为了不影响旁边学习的同学，我放下了题卷，选择了一种不容易睡着的方式，看本的确从没看过的书——《西游记》。

中国四大名著太过有名，我还识不会全书里字的时候电视剧都纷纷上演了。但看了多年的金庸小说，我知道，电视剧怎么好看也比不过原著。只是看了N回后电视剧已经先入为主，占据了我心中那四套厚书的位置。现在大了，看东西都抱一种挑刺的眼光，最近看了些《西游记》，更是失望。

《红楼梦》最先给我的印象是电视剧里那个油头粉面的贾宝玉，演不了几分钟然后大家就嗲嗲的唱上一段，有啥话一口气说完不就得了，唱来唱去不是在费咱家的电么（我承认，我不懂戏剧）？后来找书看了一段，尽是些风月之事，也许女人们会很喜欢吧。有占了中国人口一半的女性力争，《红》进入四大名著之列我也就不好说什么了。

《三国演义》出来的时候我正小，那时候讨厌历史，连带把《三》也烦上了，至今也没看几集。但高中时我爸在地摊上买了一套旧三国，绿色的，两本。我没事的时候翻了翻，很快喜欢上了它。与其说爱这书不如说爱书里的英雄，这英雄便是诸葛亮。谈笑三国的气度，空城退敌的自信，鞠躬尽瘁的忠义，悲天悯人的情怀都让我心折不已，至今仍是我的偶像。可他是个宿命主义者，且又看错了这扶不起的阿斗，让我心痛不已。唉，毕竟我不是他，他有他生活的年代，他有他的想法，说不定也还真有宿命这一说。后来看到有人批评此书一味宣扬正统，迎合统治者的喜好，我一笑了之，三国的历史和是非对我没有任何意义，管他刘备是大人奸人，我欣赏的是孔明的智慧~！

《水》是四大名著里最晚拍成电视剧的，那个时代电视制作水平已经很高，《水》自然是拍得最好的一部，音乐也制作得很好，看得我热血沸腾。可这结局是个悲剧，当初风风火火闯九州的英雄们穷途末路，一个一个倒下，看得倍不是滋味。现在我想开了，其实《水》从另一个角度上来说就是描写一帮小混混走到一起，发展成黑社会，破坏社会安定团结，最后被政府巧计奸灭的过程，看到这结局任何一个良民都应该拍手称快才行。这些只用四肢思考，杀人不负责任者个个都应该赏个死刑，甚至还有用人肉制作食品的家伙，更应该送到海牙国际法庭去判个反人类罪。

《西游记》似乎应该划分到儿童电视剧这个类别里，我从幼儿园年代直到现在读大学，每年的寒暑假都能如约的看到屏幕上那孙子跳来跳去。估计是想用这片子来教育孩子们要识善恶，要勇于和恶势力做斗争。我当童年那会也是很喜欢孙悟空的，随便拿根棍子就要舞。人们赞这猴子不畏强权，正义勇敢，是追求自由的代表，是真善美的化身。现在我大了，终于认真起来想翻翻这书，却让我大失所望。本质上说，孙悟空和上面那108流氓是一个性质，无礼法律规范和道德约束，善恶只凭个人所断，加之力量又这么强大，就是一独裁者，和希特勒相比，也就差在一念之间了。说孙悟空是自由主义者，那我们良民都得远远躲着去，他要是自由了我们就全完了。这孙悟空要不就压在那五行山上安静着，要不就取一辈子经把他牵扯着，现在这经一取完，说不定又上哪找刺激犯事去了。

再说这西游记里所描写的天界，跟现在的人间社会没多大区别，同一般的黑暗。佛家讲的是修行积善，追求心灵的完美，最后方能得道成仙到那天界去享福，料想那肯定就是传说中的共产主义了。先说说这最高统治者玉帝，如来有语：“你那厮乃是个猴子成精，焉敢欺心，要夺玉皇上帝尊位？他自幼修持，苦历过一千七百五十劫。每劫该十二万九千六百年。你算，他该多少年数，方能享受此无极大道？你那个初世为人的畜生，如何出此大言！”，可知这玉帝修行之漫长，料想修行了这么久，定是一大善之人，可再看看他表现：

1。修行了十几万年的人，看到只小猴子不顺眼就派个十万大军去缴，此为没气量。西行路上那么多妖魔，其中不乏比孙猴子能力广大者，怎么就看不到了？

2。看二郎神出场这段：”真君喝道：‘你这厮有眼无珠，认不得我么！吾乃玉帝外甥，敕封昭惠灵王二郎是也。’“。这厮一出场就搬出玉帝的关系，跟现在这些官家子子弟摆仗势欺人一模一样。这招他想必是经常用的，很可能屡试不爽，吓退过不少人，但这次碰上个不给面子的孙悟空，只好开打。这玉帝即使不是纵人行凶，管教不严也是有的。

3。下面是孙悟空几们同事的自白。
怪物闻言，连声喏喏，收了宝杖，让木吒揪了去见观音。纳头下拜，告道：“菩萨，恕我之罪，待我诉告。我不是妖邪，我是灵霄殿下侍銮舆的卷帘大将。只因在蟋桃会上，失手打碎了玻璃盏，玉帝把我打了八百，贬下界来，变得这般模样；又教七日一次，将飞剑来穿我胸胁百余下方回，故此这般苦恼。没奈何，饥寒难忍，三二日间，出波涛寻一个行人食用。不期今日无知，冲撞了大慈菩萨。”这是沙和尚。

那怪道：“我不是野豕，亦不是老彘，我本是天河里天蓬元帅。只因带酒戏弄嫦娥，玉帝把我打了二千锤，贬下尘凡；一灵真性，竟来夺舍投胎，不期错了道路，投在个母猪胎里，变得这般模样。是我咬杀母猪，打死群彘，在此处占了山场，吃人度日。不期撞着菩萨，万望拨救拔救。”这是猪八戒。

打碎个杯子的事，就施加这等烈刑，可见心胸之狭窄。八戒戏弄一下嫦娥，从另一方面说是给她在这天宫无尽的寂寞里添点情趣，说不定人家因此而心中暗喜。却给这玉帝打成这样，最后让人还再度不幸变成了猪的样子。完全可以怀疑玉帝暗恋仙子，表白遭拒，嫉恨帅哥天蓬夺得芳心，从而制造事端，对天蓬实施惨无人道的打击报复，贬其下人间后，恨犹未解，又暗中安排投胎，使其毁容，一则使其相见而不能认，二则打击他再次追求嫦娥的信心，三更是造成其种族隔离不能与人类正常生育。心之狠毒，令人发指！！

正是绝对的权力使人绝对的腐化，玉帝自己做了独裁者，自然容不下又来个挑战其权力，试想齐天的孙悟空。于是一切都有了合理的解释，以他修行十多万年的心智，自孙悟空出生便看出其威胁，又自凭现有的力量收拾不了。于是把他招安进天界让他养马来侮辱，他生气后又给他个“齐天大圣”的空衔安慰，潘桃胜会不邀，这些一松一紧的行动，就是要激怒猴子，为的是让对方先出手，然后自己就可以名正言顺的派天兵，把事情闹大，达到请得佛祖如来收拾他的目的。这一切做得不留丝毫痕迹，连招安的请求也是来自他人之口，实际上早就已暗中安排。

上头腐败如此，下面正常才奇怪了，所以那些妖怪才敢横行霸道，斗不过孙悟空了就逃回庇护伞中，可怜我们的猴子不知疲惫的只想着战、战、战……

极速快感——极品飞车9体验

发表于 2005-11-25 | 更新于 2018-05-24 | 分类于 生活

极品飞车9的发布一事最近炒得沸沸扬扬，前两代的地下狂飚做得非常成功，可以看到极品飞车系列开始已经往纯娱乐方面转型了。这9代估计娱乐元素一定不少，只知道叫《最高通缉》，估计还是和非法赛车有关。

游戏终于发布，昨天上BT搜索了一下，源很多，找了个最大3.9G的Down下来。先欣赏其中1.3G的花絮DVD，感觉剧情动画上比前两代强了不少，里面的车看起来变化还是不大，还是改得稀奇古怪的。安装盘有4CD共2.5G，比8代大了一倍，采用了更新的防破解技术，只是道高一尺，魔高一丈，破解总是有办法的。

今天才正式玩了玩。9代里可选的车辆达到了32，比8代要多不少，也走出了8代全是日本车的尴尬，世界知名的跑车都拿来过把瘾了。进入生涯模式，开始飞飚，操作的感觉和8代非常像，简单，很快就能上手。刚才也说过，娱乐性最重视，真实性就管不了那么多了。车跑起来后极度刺激，形形色色的画面极速迎来，眼睛一时都处理不过来，地图也只敢在宽点的直道上瞟一眼。当打开N2O时，画面更是带上了拖影，眼前模糊一片，不能专一的话绝对开不好。正因画面太快，游戏特意增加了一个新功能，按下极速键后，一定时间内可以把时间放慢，让游戏者可以来得急选择正确的道路和躲避车辆。我用的显卡是9550，才64M显存，开800*600的分辨率，画面细节调到第三档。虽然FPS不能做到极度流畅，但是画面的质量已经让我感动不已，光影效果做得非常棒，让我想起了WOW。

玩家在城市里参加各种非法赛车，做尽破坏，就是为了在警察的黑名单上争排名（感觉很反社会）。为此游戏里大部分的设施都可以破坏，车也变得更加稳重，不像8代里撞到棵小树也会速度减到0了。

比赛模式新增加了雷达测速赛（取从起点到终点的N个测速点得到的速度排名），加油站赛（规定时间内从一个加油站开到另一个加油站，直到终点），还有一种我就称它破坏赛吧，反正就是尽力给打破交通制度，给警察造麻烦。一方面要和其他车手竞争，另一方面要和警察抓迷藏，刺激就来了。最紧张的就是和警察的冲突，今天我试了把17辆警车追我，想尽来甩掉他们，整个追捕长达八分半钟，最后还是以我摆脱而告终，中间几次都差点被完全堵住，实在是刺激，跟好莱坞大片里一样。虽然6代也有警察的追捕，但跟它比起来就太小儿科了。

想感受真实的话我可以继续玩我的CMR2005，感受刺激的话我肯定是玩上极品飞车9了。

PS：最近常常用Eclipse来写java游戏和手机程序，感觉很不错，原来三菱有款车也叫Eclipse，当然是选它了。

Ctrl+Shift+P 定位到对于的匹配符(譬如{})
(从前面定位后面时,光标要在匹配符里面,后面到前面,则反之)

下面的快捷键是重构里面常用的,本人就自己喜欢且常用的整理一下(注:一般重构的快捷键都是Alt+Shift开头的了)

Alt+Shift+R 重命名
(是我自己最爱用的一个了,尤其是变量和类的Rename,比手工方法能节省很多劳动力)

Alt+Shift+M 抽取方法
(这是重构里面最常用的方法之一了,尤其是对一大堆泥团代码有用)

Alt+Shift+C
修改函数结构(比较实用,有N个函数调用了这个方法,修改一次搞定)

Alt+Shift+L 抽取本地变量(
可以直接把一些魔法数字和字符串抽取成一个变量,尤其是多处调用的时候)

Alt+Shift+F 把Class中的local变量变为field变量
(比较实用的功能)

Alt+Shift+I 合并变量(可能这样说有点不妥Inline)

Alt+Shift+V 移动函数和变量(不怎么常用)

Alt+Shift+Z 重构的后悔药(Undo)

再附几个:

alt+shift+s, v
覆盖/实现方法

alt+shift+s, r 生成getter和setter方法

当你按ctrl+类名跳转到别的类时，按alt +
←回到原来的类

输入sysout,按ctrl+space，自动生成System.out.println();

输入main,按ctrl+space，自动生成main()方法

想挑战吗？一起来玩Robocode

发表于 2008-01-18 | 更新于 2018-05-24 | 分类于 项目

我现在的工作趋向与业务与软件测试，离编程越来越远了，为了不让脑袋锈掉，我打算重拾JAVA，在工作闲暇之余做点有意思的项目，并为此拟了几个课题，分别是：

  1. Cbook3.0
  2. 计算机仿真
  3. 网络通信相关
  4. Robocode
    其中第一个还不具备开发条件（入手新手机），第二个比较费时，第三个还没想好具体应用，唯有第四个比较简单，而且很有互动性。

Robocode最初是IBM发起的一个开源项目，现在已经移至了sourceforge。它是一个基于J2SE、很容易使用的机器人战斗仿真器，你创建一个机器人，把它放到战场上，然后让它同其他开发者们创建的机器人进行战斗，我觉得这和斗蟋蟀很像。

最早接触Robocode是在2006年9月，当时一下就让我入了迷。我曾努力发展身边的同学一来起参与，但有兴趣的人并不多，唯一一个同学在和我打了一场后便放弃了，我也因为找不对手而没能再继续玩下去。当时Robocode还是1.1.3版，现在已经到了1.5.2，看来这一年半的时候内一直没有停止过发展。新版的Robocode中有一个roborumble.bat文件，执行它会自动下载别人编写的机器人，这顿时让我战斗欲望高涨，终于可以拿出我封尘已久的机器人和其他人过过招了。我让机器挂了一下午，下载了16M多的机器人程序，共600个左右，然后花了几天的时间，让它们以首字字母分组进行PK，视每组机器人数量，选出每组的前3-6名，准备观赏这些机器人的战斗方式来学习。以下是06年10月1日写的机器人Alpha与其中一个非常牛的机器人abc.tron3.Tron_3.11的战斗：

重游校园

发表于 2008-01-04 | 更新于 2018-05-26 | 分类于 生活

12月30日，好友某光为了增强南昌气氛、拉动南昌经济、促进南昌发展，特意从赣州赶来对南昌进行友好访问，我代表南昌人民表达了热烈的欢迎，以下是详细报道。。。

前一天加班到非常晚，11点某光给我打电话时我还睡得正HIGH，约好在北区老班办公室见面，再请老班到一觉吃饭。我晃了晃脑袋，洗漱一下便出门了。坐车到了下一站才想起没带手机，灰溜溜再回到家时已经11.半了。看看钱包里还有25块，干脆打了个车。路收又收到短信在催，我不好意思说是忘带了手机，回复了一个模棱两可的理由，比如黑社会斗殴、汽车炸弹什么的，表示今天内一定到达。一路上还挺堵，等我快到时，他们已经先去点菜了。

半年没见，这小子也没一点公务员的长进，穿了件运动型外套，跟个高中生样的，老班还是依旧那么精神亦亦。大家边吃边聊，内容境界倒是比以前高了不少，什么医院保险啊、个人所得税啊什么什么，也互通了一下各位同学的情况，有喜有悲的。

老班那天要值班，我们陪她回到了学校，合了几张影，随后在学校里逛了逛。冬日里难得的好天气，平时吃够了相机感光不足的亏，那天好好的爽了一把，边走边拍，自称“冲锋枪摄影师”。

某光、梦菲、老班、我，背景是北区行政楼，青山湖校区里最豪华的一栋楼。保安森严，用途不明，4年来极少踏入的一处地方。

某光与我，听说他最近当上了办公室主任，但又听说那办公室就他一个人。我不能理解，果然不是当公务员的料。

北区的小池塘，这边的景观毕竟还是比我们南区好得多。

九江天沐温泉游

发表于 2007-12-23 | 更新于 2018-05-24 | 分类于 生活

前阵子中测，大家忙得天昏地暗，老板答应中测结束后带大家去泡温泉，今天终于兑现了。

早上8.半在公司门口集合上车，车上的电视放了一路南昌新中原剧院的节目，时常惹得全车人哈哈大笑，但内容粗俗不堪，不提也罢。大概一个半小时后到达了目的地，我也不知道这叫什么地方，反正四面是山，还云雾缭绕的。掏出相机匆匆拍了几张，就随着大家直奔主题了。

一位MM给我们先介绍了下注意事项，之后男女各分两边换泳装，冲了便沐浴后就奔了出去。这大冬天里浑身湿淋淋走在外面，立马就冷得打抖，也不管他三七二十一，找了个人少的池就下去了。温暖满足之后都觉得泡温泉也不过如此，不就是泡澡嘛。几分钟后大家出水，分了几拔朝不同的水池一个个地试，越玩越觉得新鲜，其中有两个地方让人印象特别深刻。

一个是电子脉冲池，水下有几个可躺的地方，上面装了N多电极，人躺在这会被电得全身发麻，一阵不适过后便能感受到肌肉放松的舒爽（也可能是被电得神经麻木了），具体疗效记不得了。

还有一个是冷水池，我是第一个尝试的人，泡多了热水后骤一入其中，除了嘴里能高呼“爽~~”以外，脑袋里凉得一片空白。劝了几个同事下来试试，无一例外，得到的结论是：人真TM贱啊。。。！浸了冷水后再到热水中烫烫，据说可以舒筋活血，促进循环。

其它有意思的还有水不停转的漩涡池，像煮开水一样的气泡池，水出得像消防龙头一样的那啥XX池（记不得名字了）。中午12点的时候水上乐园开放，玩了一明一暗的两次滑道，颇为过瘾。之后又到游泳池里赛了下游泳，直到浑身乏力。

1点钟，大家团团坐在休息室中，享用着免费提供的水果饮料与我们自带的零食点心，感叹着不虚此行。我觉得连日的疲劳一招而光，其他人也都是满脸红光，个个亢奋不已。

饭后节目还是泡温泉，不过我和另外两位同事选择了要额外收费的小鱼池，早就听说有这么神奇的鱼，这次有机会不试一试就太可惜了。人一入水后，小鱼就冲着我脚丫子围了上来一顿猛啃，顿时奇痒无比，另外二位也一样，于是乎三个人一阵鬼哭狼嚎，路人纷纷顿足观看，跃跃欲试，旁边的小妹立刻粘上去：“小鱼池须要点单消费，38元一位……”。最受不了的是小鱼袭击大腿内侧、肚脐、肢胳窝这些敏感部位，那种时刻只觉得这哪里是热带小鱼，明明就是鳄鱼啊。不过泡久了后渐渐都麻木些了，三个人在看谁身上的鱼多来讨论谁比较好吃一些……反正我是没他们好吃啦，估计是昨天才洗了澡的缘故吧，没他们身上这么多佐料，哈哈~~

到了集合时间后回到更衣室穿上衣服，掏出手机，人人都有未接电话，号码大多来自广西，要知道咱们在这大声叫爽的时候广西的同事们真焦头烂额地搞中测，这些电话肯定是找咱们要帮忙的。回忆咱们中测时着那噩梦般的日子，心中的舒泰立马又上升了一个等级……

归来的路上，汽车依旧在放未完的节目，但心情已太不一样，唯一的遗憾就是泡温泉时不方便带相机，没留下最有意思的那些画面。照片改天再补上来，人先去睡了~~~~

公司圣诞活动

发表于 2007-12-22 | 更新于 2018-05-24 | 分类于 生活

昨天下午干活的时候我键盘敲得飞快，因为下班后是公司一年一度的圣诞活动，可不想拖得太晚。以前常听同事们谈起往年的趣事，个个都津津乐道，这次终于能见识下了。

虽然很尽力了，还是耽搁了点时间，等我来到预订的饭店时，大部分人都已经坐好了。如往年一样，参加的还有移动计费中心的同事们，听说有15桌，我没仔细数。找到位置坐下后环顾了下四周，看到了很多同事家属与“家属”，很是地满足了一下好奇心。桌对面李道长在掰他这次的北京之行（跟我上次一样，去参加新员工培训），主要内容是他和美女同事不得不说的故事，引得大家唏嘘不已。 菜慢慢端了上来，计费中心的两位老总以及我们的BOSS讲了几句话，往下便是每年一度的重头戏抽奖了。大家的名字都在一个箱子里，从三等奖开始，一直到特等奖，念到名字的上去领。据说也是惯例，先上台进行抽奖的要是这一年里新出的一对情侣，于是计费中心的花花便和他的GF扭扭捏捏上了台。我有幸中了个三等奖，100块的洪客隆现金卷，二、一、特等奖分别是挎包、电磁炉和浴足器，一时间有人欢喜有人忧，气氛渐渐上了高潮。最终没被念到名字的人是四等奖，可选电暖垫、颈枕、厨用刀具等，也挺不错。

饭后的节目是去旁边的沃尔玛四楼中影看电影，在《投名状》和《集结号》里自由选择，电影票中午就发到了大家手中，我选的是《集结号》。

以前也来过中影看过几次，一般都是冷冷清清的，从没向这次一样这么多人，难怪电影都想赶着圣诞上映，节日期间的民众真是巨舍得花钱啊。

片子基本没让人失望，开始是战争片，这战争场面绝对是中国战争电影中真实与残酷No.1，比起《夜袭》给人话剧一般的感觉那是一个天一个地。许多场景看起来让人立刻回想到《拯救大兵瑞恩》（感觉有模仿痕迹），但那个镜头时常晃啊晃啊让我脑中总是只有些片断，把握不到到整个战场，虽然知道这是为了突出那种紧张不安的气氛，但未免太过头了。之前看了三表哥谈《集结号》，说这片子音效糟糕，我倒觉得还蛮好，虽然不幸坐在了影院的左后角。

几阵激烈的战斗过后，士兵们都死光了，只剩下连长莫名其妙地活了下来，往后就基本没什么战打，变成了剧情片，主要靠煽情了。主角的演技很不错，挺让人感动，可能主要还是我心软。

明天公司组织去泡温泉，期待。。。

其实我还健在

发表于 2007-11-16 | 更新于 2018-05-23 | 分类于 生活

这篇博文没有主题，只是告诉各位光临的过客，我至今仍然健在。我也想为这段时间的无作为举点理由，但嫦娥登月我错过了登机、华南虎那会我相机没电、拍《色戒》赶上我没档期、制定新假期制度时我又在加班，于是……

还是汇报一下工作吧：

加班天昏地暗

最近公事N多，升级/故障/需求/中测，压得喘不过气来，频繁地通宵。整得白天瞌睡眠眠，夜晚精神亢亢。好不容易躺下，还做梦梦见电话在响，惊醒后发现电话果然在想。。。额的上帝老天爷啊！

消费日日高攀

以前读书时穷惯了，省吃俭用，但也积累了不少数码产品。现在经济终于宽松了点，除了买了个相机看得摸得着外，不少Money都入了肚肠。一边在宿舍交足了伙食费，一边三天两头的在外吃饭，不是我请同事就是同事请我，出门抬手就拦车，付钱时还假惺惺要张发票。

基金石沉大海

平日里不少同事在玩基金股票，耳濡目染受多了影响，又因为是牛市，也有些跃跃欲试。 10.11号正好广发聚丰开放申购，试着买了1000块，没想到卖得太火。最终配额四份之一，于是我的第一份基金成了250，当时大盘是6000多点。后来又入了几K其它两种基金，结果连续赶上几波大跌，前两天竟然探到5100左右。NND，时不待我啊，基金竟然都能被套牢。还是眼不见为净，尽量不看，心中却日日盼涨。

心中拔凉拔凉

生活这么没劲，心里自然爽不到哪里去了。最近也没出什么好电影好小说让人以寄托，每天就是打开抓虾欣赏下各位牛人的博客，找点冷笑话自讨开心而已。现在就盼着中测能顺利度过，还我以正常生活。或者时间有多还能拾起编程旧事，写点有意思的东西。

料理鼠王(Ratatouille)

发表于 2007-10-20 | 更新于 2018-05-23 | 分类于 扯淡

我一直认为，皮克斯之于动画就像暴雪之于游戏——一旦出品，必属精品！

昨天晚上挂了一通宵，终于下到了皮克斯制作的新片《鼠王料理》。不知道是最近关心太少还是迪斯尼没花大力宣传，我事先竟然不知道一点有关这部片子的消息，还沉浸最近的《汽车总动员》之中没缓过神来。

故事发生在美食之都巴黎，讲的是一只非常有烹饪天赋的老鼠和一个非常没有天赋的厨师，老鼠藏在厨师的帽子里，提着他的头发像操作提线木偶一样帮厨师烹饪，最终成就了最无可挑剔的美食。

看完了这部片子后，没有期待中非常过瘾的感觉。我尽力让自己以为这是我胃口太刁所致，不愿意承认皮克斯终于也有力不从心的时候。《鼠》画面上不如《超人总动员》和《汽车总动员》绚烂，剧情感觉老套，创意也没太多吸引人的地方（还记得《蚂蚁总动员》中那只小蚂蚁用露水和叶子造的望远镜么？）。

因为某些原因，我们的神州行反复下移推迟，但今天我们的办公室给下移了——从原来网管大厦的第9层搬到了另一栋小楼的第4层。

还是家好

发表于 2007-10-06 | 更新于 2018-05-23 | 分类于 思考

10月3号晚上，漫长的三个半小时车程之后，我回到了萍乡。以前一直生活在这，没觉得有什么变化，上大学后几个月才回来一次，终于承认每一次我所见的萍乡都是新的。

老妈仍旧是在门口等着，Taxi一停下就迎了上来，比上次见她又胖了点，我没说，知道她不喜欢。见家门后看到了外公外婆，仍然神采亦亦。表弟的变化比萍乡快，现在已经是个小伙子了，仍旧不太爱说话。他的妈妈，也就是我舅妈，还是那副笑盈盈的样子，顶着大肚子。据说预产期在7号，我希望这次能多个妹妹。打遍招呼后开始吃饭，虽然辣椒比我希望的少，毛豆比我预料的硬，但给我的感觉还是塞过吃酒店。吃这一点我还是比较容易满足的，但又最难满足，因为除了萍乡，其它地方做菜都不怎么放辣椒。

第二天陪妈去逛了下街，给家里餐桌置了块漂亮的桌布。她说最近心绪不好，我们到药店挑了瓶药，140块，趁她掏钱时我抢先付了，营业员羡慕的眼神下，我知道妈很欣慰。工作半年了，能用钱买到爸妈的笑容，累点也值。

晚饭上终于吃到了足够份量的辣椒菜，大呼过瘾，掏出相机拍了三张，现在看看还口水直流。有些东西，真的是无法改变，也不愿改变。

两天的生活很简单：看电视剧《士兵突击》和《丑女贝蒂》、吃饭、睡觉。今天大表弟来了一趟，这两年他一直在外面打工，都没见着面；如今回到了萍乡工作，还诓了个MM回来，我妈借题没老少唠叨我找女朋友的事。

好日子总是过得很快，明天又要回南昌了，没意外的话，下一次回来就是过年了。

最初让我选择R6的就是它的强大的性能参数了，28mm的广角镜头，7.1倍的光学变焦，1cm的微距，如此多的特性集中在一个小巧的机身里，实在是不可思议。在众多同水平的DC中，R6是独秀一枝的。下面我以我的体会，谈谈R6的优缺点与使用心得。

外观与操控：

我买的是黑色的版本，如果只能给一个评价的话，那就是很酷。近一个月的使用上来看，还没有发现什么外观设计得很不合理地地方。虽然参数里标称机身最薄处是20.6mm，最厚处23.3mm，但整个相机除了右侧的握手端较薄外，其它地方都是算“最厚处”，而且不是标称的23.3mm，实测是应该是25mm，因为前在前方的镜头关闭后仍然突出了机体1.5mm多。要想当名副其实的卡片机的话，应该再瘦个8mm左右才差不多。但在实际出行中，25mm的机身携带还是很方便的。机器的菜单操作比较繁琐，找有些东西要按下MENU后要一页一页地翻。但如果只是把它当傻瓜机用的话，基本的几个按钮足够用了，学会灵活使用ADJ键就好，我的ADJ配置依次为曝光补偿、白平衡、ISO设置和测光模式。

★★★优点★★★

广角：相对于大多数DC的35mm短焦，R6的28mm很有优势，广角镜头在拍一些近或者大的建筑时有更强的视觉冲击力。最简单的一点是，人起码不用挪到很远的地方才能拍出建筑的全景。在实际使用中，广角端带来的便利是大大超过长焦端的。

长焦：R6号称7.1倍的光学变焦只是相对于它28mm的广角端来说的，长焦端200mm的焦距在普通的DC上也就是5.7倍的光学变焦。但比起大多数机器的3倍，这也是非常强的了。在实际使用中，要用到长焦端的机会并不多，并不用对此太过在意。

微距：该机的微距功能很牛，在室外拍花草的话想凑多近凑多近，至于室内的话，还是先想好怎么布光再说吧。 但是R6是全自动机，在微距模式下拍某些东西可能会难以正确对焦。

快门1/68秒 光圈F3.6 焦距5.4 ISO100 曝光补偿0.0 最大光圈F3.1

续航：参数上写着理光的续航能力是330张，但在我的实际使用中一块原装电池充满电后拍了440张才自动关机，另外一块品胜的备用电池也拍了300张照片。这些照片全是最佳质量模式，部分使用了闪光灯，还有几次开机全部浏览以前拍的照片。这一点非常让人满意。

☆☆☆缺点☆☆☆

噪点：机器的最高ISO有1600，但我从未使用过，因为它在ISO200的情况下就已经够让我失望的了。在光线稍微暗一点的室内，拍出的照片就经常惨不忍睹。这是该机器最不能容忍的缺点。

LCD：机器背面的LCD屏不小，但显示效果并不是十分满意，总感觉画面比较闪烁。而且当其中某部分很亮时，它所处的垂直方向会出现一个亮条贯穿整个画面。只能说还凑和吧！

变焦： 机器的变焦不灵活，从拨动变焦环到产生变焦动作有半秒左右的延迟，这使得要精确的变焦很不容易，只能多试几次，特别是从长焦变短焦时延迟更长。
机器变焦的声音很大，特别是在微距对焦的时候，机器会发出嘎嘎嘎的声音，我都很长久以后会出故障。
实际拍照中变焦使用并不很频繁，上述问题对使用的影响并不大，但就是有些心里不舒服。

镜头卡死： 比起前面这三个缺点，这个简直就是噩梦了。想想在玩得正High的时候，朋友在前面摆POSE等着你，而此时你的机器镜头卡住了，人尴尬不说，旅游的心情也因此完全被破坏了。我就经历过一次，幸好被我摆弄好了，待我在下文中再分享心得。

★★★心得★★★

我想任何一台相机都有它的优缺点，能拍出怎样的照片很大关系上是与使用者有关的，我们应该在使用中尽量扬长避短才是。

R6的噪点多，想拍出少噪点的PP，首先就是降低它的ISO。在光线好的时候使用ISO200没有问题，在阴天应该尽量使用ISO100，至于晚上还是乖乖用ISO64吧。

ISO降低了快门速度自然会降低，这就要求在拍照过程中手要更稳，R6虽然有光学防抖，但效果并不是很好，这时候已经帮不上你什么了。这时候有两种解决办法：一是多拍几张，对于风景或建筑这点很容易，如果面前站的是个人的话，你还是快点吧，不然表情都走样了；二是使用三角架，这是个矛盾的办法，我想没谁会为了一台准卡片机随身背一个三角架。作为替代，我们可以随便找一个能放稳相机的地方，对好目标后使用2秒的自拍模式。我为了拍好晚上的街道，就曾把机器放在垃圾筒上这样用过。下面这张图是放路灯底座上用自拍模式拍的。

关于镜头卡死：镜头卡死有时是因为没电了，特别是在用非原装电池时可能会在变焦的时候突然断电，这种情况下换个有电的电池就好了。

第二种情况可能跟操作有关，我记得我当初就是在照片浏览模式开机（就是长按浏览键开机）切换到拍摄模式时并迅速关开机导致的镜头卡住。此时镜头凸出机身约2mm，闸式镜头盖无法闭合。关机后再开机，LCD也不亮了，镜头仍然不动，机器里传来喀喀的声音。当时我心中拔凉拔凉，传说中理光R6镜头卡死的故障怎么偏偏让我碰上了，而且偏偏是在我出来玩的时候。因为LCD不亮，搞不清机器到底是开着还是关着，我尝试着取出电池，再装回去开机，仍然没有效果。我怕镜头变焦马达烧坏，遂取出了电池另想办法。我是在淘宝买的水货机器，要保修还得寄回商家处去。当时连暴力破解的想法都有了，后来灵光一现：平常按相机开关时镜头会自动伸出至待命状态，而长按照片浏览键开机时镜头不会有动作。那在镜头伸出的情况下按浏览键开机，机器会不会试图把镜头缩回呢？装上电池一试，一阵喀嚓喀嚓之后相机竟然恢复正常了，谢天谢地！当时实在太激动，也忘了具体是怎么变的来着，有R6镜头卡死的朋友在没办法可想时可以尝试一下。

终于挤出时间整理完了照片并上传到了yupoo，现在开始要再挤时间结合照片把这次北京之行所见所感写一写，毕竟是一次难忘的经历。

第一项是互相认识：给每个人发一张A4的白纸，在上面折出36个同样大小的格子，在第一个格子写上自己的名字和两个形容自己特点的词语，然后在15分钟内去认识35个人，并请他们把自己的姓名和特点填在白纸上。一声开始后大家就忙开了，到处握手，互相介绍。我这人老没记心，到后来找到几个人都是已经签过名字了的，然后道声抱歉再继续找。虽然最后时间截止时我还差5、6个格子没填上，但还是觉得收获不小，见到了几个以前在工作中打过交道的同事，介绍一番后大呼：“原来你就是XXX啊”。

第二项是组队：55个人随机分成三组，要各组在30分钟内确定好队长、队名、队标、队歌、口号、队形。这其实挺不容易的，最后三支队分别是星之队、翔之队和我所在的烈火雄心。

我们在清华西门下车，感觉这街比我学校西门口的上海北路还破，相同的是，我见到了带着洋红色霓光的房子，嘿嘿。清华里面古色古香的，看到了朱自清先生笔下的荷塘（这荷塘太多了，也不知他当年写的是哪块）。校园很大，我和文明君就是绕来绕去地瞎逛，最后从东门出来。清华美女并不是很多，见到了几个吧，再说晚上也看不太清楚。我们在东门口找不到吃饭的地方，无奈打个车回到西门找了大排档。

17号安排参观公司。我和文明君怕堵车，很早就上了运通105公交。果然不出我们所料，从宾馆到西直门那段，原来10分钟的路程走走停停，硬是磨了半个钟头。

公司跟想象中差不多，占了中电大厦四个楼层，每一层都挺不小的，就是天花板有点低，比较压抑。我估计在这上班没我们南昌开心，我们那就几十号人，大家都熟得很，在办公室里很自在，可惜再不久就要搬了，听说新办公室挺烂的。

　　从公司出来后我们江西4个人再加其它省的4人去逛颐和园。颐和园联票票价60，我们还请了个私人导游给我们一路介绍。这园子很大很漂亮，不知道慈禧老太太怎么能逛得完，里面绝大多数景点都是被八国联军破坏后重建的，这让人感觉非常爽，不知道那些外国游客怎么想。里面景点我记不太清了，直接上图片吧。

—————————现实的分隔线 —————————

这次北京之行，我收获还是颇多的。见识了一下首都风情，感受了一下京城人民的生活环境；体验式培训与“成功企业人”两课给人印象深刻，还得以后在工作中慢慢消化；逛了下颐和园，让最近紧张的情绪得到了最大的放松；狂拍了700多张照片，过了把摄影的瘾。

北京很大，人很多，所以交通很乱、很慢。如果每天要花几个小时在上班下班路上的话，无形中降低了人的生活效率，生命因此而削减，这是我非常难以接受的。当初也是因为这一点，我不太愿意去广州、深圳、上海这些地方去工作。至于北京的空气，还是算了吧。。。

以前一直以为北京是个人文城市，有很多历史积淀。这几天下来，我更觉得北京是个人造城市。那些历史古迹大多被八国联军破坏得不成样子，现在能看到的大多是后来翻新或者重建的成果。现在没有了八国联军，但有了世界各地的游客，于是能看到天安门上的有机玻璃罩和颐和园铜塑上的铁丝网这些不伦不类的东西，于是北京城仍然在不断的翻新与重建。除了那些百年的老树外，其它我所看到的我都不能确定它是否真的是历史。

我对北京人的印象还是很好的，在北京多次的问路都能得到礼貌而详细的回答。北京人很客气，特爱说“您”，让人很温暖。都说北京的的哥能侃，就我打的的几次情况来看，司机都很沉默，难道都是外地来北京打拼的司机？

来北京不得不说奥运，在北京可以看到很多卖奥运纪念品的商店，我常常能被无意中发现的一句“One World, One Dream”所感动。为了迎接奥运，北京一直在努力改变着，到处是施工的公路与建筑，这使得在我面前出现的是一座“破”城。希望2008年的北京，会是一座现代、环保、包容的城市，给全世界人一新的感觉。

最后还得感谢公司组织一次这么大规模和规格的培训，让我们新入职的员工感受到了公司的温暖。

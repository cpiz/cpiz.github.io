又搬了一次家

发表于 2007-04-27 | 更新于 2018-05-24 | 分类于 建站

blogger本是个好东西，但顶不过GFW，动不动就歇菜。不希望自己的博客再这样被别人左右，还是放自己的空间保险，多花点钱也无所谓。

在pjblog和wordpress之间犹豫了很久，最后还是选择了更简单漂亮的pjblog。因为blogger迟迟不解封，等了最近恢复才把数据备份了下来。pjblog不支持直接从blogger搬家，最后把数据从XML转入ACCESS里才完成导入，但以前文章的格式丢了好多，图片也没了，待慢慢补上。

pjblog功能比google blogger强大了很多，因为空间是自己的，想怎么改就可以怎么改，有很大的灵活性，而且pjblog的插件也不少，希望这是最后一次搬家吧。

上班第一天

发表于 2007-04-03 | 更新于 2018-05-24 | 分类于 工作

糊里糊涂通过了亚信的笔试和面试，对方3月26号通知我从下周一（4月2号）开始实习。于是在这6天里疯狂的做毕设，希望先把代码搞定，留着论文有空再说。虽然每天都努力到三四点，但最终还是没有搞定，还差了敌人的AI没写，现在敌人都傻傻地站那等着我过去杀。

４月１号１１点半就爬上了床，可是因为近来生物钟的关系，听着外面风声呼啸到了三点多才能睡着。第二天早上７天起来迷迷糊糊的，生怕落了什么东西没带。外面仍然在刮风下雨，买了几个包子一袋牛奶边走边塞，另一只手还得撑着伞，结果牛奶撒了一裤子。
我算比较走运，刚到车站就来了公车，在谢家村站上车的另一位和我一起的同学说他都等二十分钟了。车很挤，又是上班高峰期，边走边停半小时多才到，本来时间刚刚好，在楼上登记临时出入证的时候拖了几分钟，到办公室的时候已经迟到了。我们的办公室是公司放服务器的机房，几里Ｎ台ＩＢＭ的服务器，风扇吼吼地响着。我们机器是从公司以前报废的机器里淘出来的，一台Ｐ４，一台Ｃ１.1，还有两台Ｐｉｉｉ　８００的笔记本。公司有网络，但我们办公室接的是个１０Ｍ的ＨＵＢ，能上ＱＱ和ＭＳＮ，但不能看网页。
这几台机器捣了我们一上午，中午是在移动食堂吃的饭，５块钱一餐，味道相当不怎么样。下午和带我们的同事分别见了面，给我们发了很多文档，让我们先看看来熟悉业务，很快第一天就这么过去了。
下班回去的公交更挤，折腾到６点半才回到学校，钟妹妹说这是我第一天工作，磨着让我请她吃饭。好不容易打发她走，到外面洗个澡，满身泡沫的时候手机又不停地响，让我还以为是宿舍失火了，却是胖子哥托办点事。回到宿舍后胖子哥已经在门口等了十来分钟了，跟他谈好了事情后又被室友拉去帮忙弄毕设，折腾到１１点就躺倒，挨上枕头一会后就睡着了。

改进了查找的算法,搜索更快了一些
更改了跳转模式,通过滑动条或直接键入百分比达到文本位置,步进达到了1%
修正了文本有的位置出现一个方框符号和换行的bug
修正了偶尔锁定键盘时背景灯不同步的bug
增加了行距调整功能,减少了看书时的疲劳感
改进了文字显示算法,翻页反应速度更快
改正了长方形屏幕应用横向持握状态后显示不正常及报错退出的bug
其它等等…

按<*>键锁定与解锁键盘,键盘锁定后背景灯随之关闭.一分钟无操作自动锁定

拨开云雾见天日

发表于 2007-03-10 | 更新于 2018-05-24 | 分类于 生活

自过年后心情一直比较低调，想到开学后要进行的离散考试就郁闷无比。放寒假前从学校回家的时候忘了带离散课本，于是原本想在家复习的计划也泡了汤。

3月2号拿到了准考证，得知考试被安排在9号，稍稍缓了口气，感觉像是一个得到缓刑的犯人。然后接下来的几天都比较忙，因为一直在思考：看书，或者不看，这是一个问题。考前两天经N方渠道打听到了这次考试的出题老师，和两个同样不幸的同学磨了他两天，终于打听到了一些题目。

考试相当顺利，题目都没出老师勾划准备的范围，而且还变成了开卷，于是把原本打算偷偷抄的两页纸堂而皇之拿出来copy，十分钟就搞定，在草稿纸上练了半个钟头书法才交了卷。

走出考场的时候我真想大吼几声，最终还是憋到了宿舍才出声，三年的积郁倾刻消散，终于拨开云雾重见天日了。自从大一下学期挂了高等数学，大二上学期挂了离散数学和数字逻辑后，每学期的重修考试前我总是茶饭不香，交了700多重修费不说，不知道浪费了多少时间，耽误了多少项目。这三年来，我都数不请我这三门课重考了多少次，又挂了多少次，到今天终于被我一一消灭了。

考试这块石头落地后，我终于可以全心地投入到毕业设计和找工作的事务当中。寝室里有两个室友一个走深圳，一个去广州，已经先一步出去了，去印度实习的同学也出发了，巧的是他们都是在我考离散的时候离开的。

寝室里少了两个人一下冷清了好多，我又躺回了床上过去了猪样的生活。阿贵跑来通知我他下个月１８号结婚，问我有没有空，这应该是我们这届最早结婚的了吧，祝他们幸福！

2007聚会一

发表于 2007-02-25 | 更新于 2018-05-24 | 分类于 生活

初一到初四家里来了好多亲朋，初五是小学与初中同学聚会。除了几个一起读到了高中甚至大学的同学外，很多人都是多年未见。大家变化了很多，曾经不怎么会说话的如今油嘴滑舌，曾经穿着保守的如今花枝招展。一见面就是扑克和麻将，男人们谈的是玩乐和女人，女人产谈论的是玩乐和男人，让我极不适应，只能和旁边两位读过大学的同学聊得起来，话题是工作和前途之类。

因为组织不当，晚饭没有提前预订，跑了N处档次高点的饭店都没有位子，不禁感叹一下萍城人民的消费能力，最后还是找了一个家乡菜馆凑和了一顿，环境不行但菜还不错。

吃完饭照例是K歌，某些同志破锣嗓吼得天响，我自知五音不全藏拙不上，大功率的音响相当嚣张，和旁边人说两句话也得对着耳朵狂喊。

最大的收获是后来见到了小学的班主任王老师，对王老师我是又爱又恨。王老师教我们语言，课讲得很不错，对更是我相当欣赏，但她布置的作业极多，又喜欢体罚学生。我那时候是班上不做作业的典型，挨打是家常便饭的事，数不清流了多少眼泪。然而我又是屡教不改的典型，于是这一打就是整个小学五年。小学毕业时我心情好比四九年全国解放、孙悟空被救出了五指山，终于是熬过来了（没想到后来还是被初中的英语老师凑了一回，那是题外话了）。她虽然没给我落下什么后遣症，但落下个心里阴影是绝对有的。

再次见到王老师她已经不是当年我们面前那副飞扬跋扈样了，而是一副标准的和蔼妇女形象。我问老师还记不记得我叫什么，她说她忘了名字，但是记得我老是不交作业，但是考试起来总是九十多分（汗！）。我和老师聊了一会，谈起了我小时候挨过的打、我闯过的祸，老师也依然记得很清楚，她也谈起了当年对我的欣赏，对我的恨铁不成钢。我感谢王老师，感谢她对我的赏识，感谢她当年对我的严格，很多当年不能理解的东西现在也昭然而释，正如她所说：“老师是最无私的”，老师把她最宝贵的年华花费在了我们的身上，为我们操心烦恼，不懂事的我们却不停给她制造麻烦，我们实在欠她太多了（但我对体罚的教育手段保留看法）！

向大家告了别，给老师鞠了一躬，我离开了。

年

发表于 2007-02-25 | 更新于 2018-05-24 | 分类于 思考

每当要开始写这种总结性质的文字的时候，心中总是会涌上一番时光如梦的感叹，等回过神来，原本打好的腹稿已无处下笔。

三十那天上午依旧是父亲带着我到老家为爷爷上坟，流程依旧是上香上菜烧纸求福，然后放一副爆竹收拾下山。我本不相信这世上有鬼神，但如果爷爷在天之灵能佑我全家的放在，我是宁愿相信的。

上完坟后去看望奶奶，奶奶今年已经92岁了，十几年来本一直住在我家，去年在伯父家住时不幸摔断了盆骨，落下个生活不能自理。我在她浑浊的眼睛中只看得到迷离，父亲跟她说话的时候她只能无力的应上两句，我怀疑耳背神离的她是否听清了父亲在问什么。说话间父亲的声音带上了哽咽，一直沉默在旁的我心中一阵刺痛。别人在羡慕奶奶寿命的时候不会理解这种年龄的负担，外面是热闹的世界，奶奶只能在这冰冷的房间中躺着，这已经是第五个月了。我难以想象让我毫无希望地在床上躺四个多月会是一种多大的痛苦，大概我会早忍不住想办法终结了自己的生命。

奶奶的事给我们全家带来了一层阴影，势利的伯母因赡养奶奶的事多次与父亲和姑姑发生矛盾，更年期的母亲神经脆弱，为这事也没少和父亲吵过架。我曾建议父亲诉之以法律，还被重情义的父亲批评过，说好歹是亲生的兄弟姊妹，不希望从此变成陌生人，我是独生子，对此不太能理解。

这一年看到父亲老了许多，做儿女最大的痛苦莫过于此（想起了朱自清的《背影》）。

往年过年都是我们一家三口和奶奶在自己家，这次的新年是在外公外婆家过的，除了一个姨妈和表弟外，母亲这边的亲戚都到齐了，气氛热闹得多。贴春联放鞭炮，三代同堂吃了顿和谐的晚饭。晚饭后亲戚们都到我家来玩，我今年甚至没看春晚，和他们玩了几个钟头的21点，倒也其乐融融。快到午夜的时候外面开始爆竹隆隆，焰火满天，我突然想看看这时候谁会在上网，没想到仍然看到了大堆的同学朋友。

3D之家

发表于 2007-02-09 | 更新于 2018-05-24 | 分类于 技术

回家快十天了,一直在忙项目,连QQ和MSN也几乎没上.每天两三点睡,早上十一二点醒,除了生活质量高一些外,没感觉有一点放假的轻松.只怪自己当初估算得太过简单,没做好项目估算和风险评估,实在是有愧于项目管理老师.

前两天在网上找软件的时候发现了Google SketchUp,介绍上说这是人人都会用的3D制作工具,本着对Google的好感下载了一个试试.软件版本是6.0.312,绿色解压版.打开后明显能发现Google一贯的风格,界面也比较简洁,只是E文的菜单让我靠着金山词霸查了半天才知道大概作用.

给自己找了个题目:制作自己家的3D模型.这软件的确简单,个把小时后已经用得比较熟练了,N个小时后完成了本人这第一份3D作品,之间我拿了个卷尽把家里量了个遍.

由于软件里可用的建筑模型有限,很多家具都画不上去,不过还是相当满意了.最后还使用软件制作了一段动画,让我脑袋被3D画面晃得晕晕的.

演出结束

发表于 2006-12-28 | 更新于 2018-05-23 | 分类于 生活

折腾了N天，尽管很不情愿，但还是一直在参与着音乐剧《暖暖四年》的排练。今天，终于圆满谢幕了。

多请了半天假，偷个懒多睡了半天，总算调整了一下最近紊乱的生物钟。起床洗漱吃饭后开始准备我要用到的道具和服装，有意思的是，我上台时穿的那一套行头没一样是我自己的：A的皮鞋，B的裤子，C的皮带，D的毛衣，E的西服。看来我平常穿着相当失败，是该去添置点了。

从下午一点多到场馆准备，直到四点多才上台彩排了一次，大家表现得相当失败，让所有人都开始担心了起来。仔细查漏补缺了一番，都饿得半死时才弄来盒饭，草草填了个肚子。

七点三十分，晚会终于开始，本来想随便演下便过去了，不得已还是被抓着画了下妆。我们节目排在第十五，也是倒数第三个，中间我和另三人反复练习了一下朗诵与走位，到要上台时，累得嗓子冒烟，腿也站得发酸。

真正开演了，可以说一切顺利，舞台效果也很好，我演的从被拒绝到喝愁酒到相恋这一段台下的观众都有很大的反应，后面的朗诵也没有背错词。最后近三千人的掌声让人真的很满足，正如我们所演的，暖暖四年过去，我们就这样走了，这是最后一次踏足校园晚会的舞台。

回到台下，同学们纷纷对我们的节目表示了赞赏，让人相当的欣慰，这阵子的努力总算没有白费。终于可以一心一意地做项目，中午可以一心一意地玩玩SC，晚上一心一意的休息了。。。

累了一天，睡了~晚安，世界！

我希望[转]

发表于 2006-12-16 | 更新于 2018-05-23 | 分类于 文摘

我希望人们常问，应该找一个什么样的伴侣呢?
我说——
我希望，她，和我一样，
胸中有血，心头有伤。
不要什么月圆花好，
不要什么笛短箫长。
要穷，穷得像茶，
苦中一缕清香。
要傲，傲得像兰，
高挂一脸秋霜。
我们一样，就敢在暗夜里，
徘徊在白色的坟场。
去倾听鸱枭的惨笑，
追逐那飘移的荧光。
我们一样，就敢在森林里，
打下通往前程的标桩。
哪管枯枝上，猿伸长臂，
何惧石丛里，蛇吐绿芒。
我们一样，就敢随着大鲸，
划起一叶咿哑的扁舟，
去探索那遥远的海港，
任凭风如丧钟，雾似飞网。
我们一样，就敢在泥沼里，
种下松籽，要它成梁。
我们一样，就敢挽起朝晖，
踩着鲜花，走向死亡!
虽然，我只是一粒芝麻，
被风吹离了茎的故乡。
远别云雀婉转的歌喉，
远别玫瑰迷人的芬芳。
我坚信，也有另一颗芝麻，
躺在风风雨雨的大地上。
我们虽未相识，但我终极乐观，
因为我们顶的是同一轮太阳。
就这样，在遮天的星群里，
去寻找那粒闪烁的微光。
就这样，在蔽日的密林中，
去辨认那片模糊的叶掌!

刚在淘宝上订了一台LINKSYS 11M无线路由器，几天后就能享受无线了。

最后的决战

发表于 2006-07-10 | 更新于 2018-05-23 | 分类于 生活

凌晨一点半，关了电脑，还是顶不过瞌睡一头栽向了床。躺了半小时都没睡着，心里还是惦记着德国的决赛，一直把自己定位于伪球迷，今天不会是要转正吧？一个鲤鱼打挺跳起了床，刚打开电视机，正好上演齐达内的勺子点球。至于什么是勺子点球，具体我等水平也不太明白，总之这球T得很赞，老黄都把人齐达内吹到天上去了，就差周星星附体这等惊句了。比赛刚刚6分钟，1：0，让我心里稳当了些，法国队这次该拿冠军了吧。

我等伪球迷除了专业不精以外，阵脚不稳也是一太特点。世界杯没开始的时候希望巴西赢，因为喜欢巴西的技术风格。看了阿根延对塞黑的6:0那场，对阿根廷屁服得五体投地，23连传的世界波让人颇开眼界，大有人挡杀人佛挡杀佛人之势，马拉多纳在主席台上乐得屁颠屁颠，谁知被德人杀得……不说也罢。然后指望英格兰，不料与葡萄牙一役，小贝受伤鲁尼使坏，个个不争气，点球大战1：3遭汰，事后小贝宣布不再担任英格兰队长，估计不少MM要伤心一阵。一直投以最大关注的巴西除了肥罗进了他世界杯第15球破了记录外，其它表现非常让人不满意，小罗太让人失望。巴西遭法国淘汰后CCTV5上一直还在播小罗给联想做的广告，让人直接怀疑联想本本的质量。

回到决赛赛场上来，19分钟意大利得到角球，马特拉齐头球将球得分将比分掰平，注定这将是场精彩的比赛。然后是齐达内与亨利的SHOWTIME，逼得意大利数次拉响警报，防线岌岌可危。老黄再次证明了他中国第一足球解说员的能力，妙语连珠，什么保险箱防盗门灭火器全来了，让旁边的张路只有不断嗯啊应和的份。意大利的运气实在大好了，死撑苦撑到了加时赛，然后发生了戏剧性的一幕。马特拉齐不知道说了句什么激怒了齐达内，齐祖大展铁头功，将1米93的马特拉齐一头撞倒，实在是干净利落，堪为铁头扬威的教科书。场上的裁判都不知发生了什么事，半天后主裁才掏出红牌将齐达内罚下了场。事后才知道是场外的第四裁判看了电视录象，然后用通讯系统告知主裁，此举称为足球史上第一次依据电视录象的判罚。告别球坛的最后演出，一代球王就此惨淡收场，齐达内走进休息室与旁边的金杯擦身而过的镜头让人心酸不已，这是否真的意味着接下来的比赛将不再属于法国？

点球大战在即，法国队差点找不到5个可以射门的人，最终还是3:5落败无缘金杯，不知身处休息室中的齐达内那时作何感想。如果齐达内在场，点球未免就会输，甚至可能不用拖到点球时刻。嗟呼，成也齐祖，败也齐祖。

六月求解

发表于 2006-06-19 | 更新于 2018-05-23 | 分类于 生活

六月 = 考试 + 毕业 + 世界杯 + X

###六月是考试的月份。###

大一大二的六月是一年里最紧张时间之一，逃了一学期的课程不得不在这30天里统统补回，顶着酷热开始学期第一次自习，琢磨老师的出卷范围，借钱MM的笔记去复印，不停后悔错过了哪节重要的课程……今年六月是个特别，重要的课程都以大作业或实验报告的形式在五月份就搞定，剩下的只有些很容易就pass的科目，让人颇觉得舒爽。在老同学面前炫耀一番，感觉自己牛B烘烘，似乎已经看到成绩单上门门上百。

六月唯一正式些考试也只有英语四六级了。面对这大学生最可恶的敌人，闭目待死者有之，忐忑不安者有之，终南寻径者有之，浑浑噩噩如我者有之（幸好我是考六级），一场过后几家欢喜几家愁。像我这种单词只背到abandon而又奇迹般过掉了四级者，只是抱着拿份六级成绩单的目的来报的名。看到试题后大呼后悔，只想把卷还给监考老师再让她退还我35块报名费。除了听力外，其它题都是既不认识题目也不认识答案，挑了些看着还顺眼的选项勾划上，草草写了篇作文，还提前了几分钟交卷，得来一片诧异目光。不乏友人问及感觉，答曰：“填空尚可”，复问：“填空何处之有？”，答：“姓名与准考证号”。

三年前的六月我参加了第二次高考，迎来人生第一大转折，结果是来到了这里。三年前的今天看到网上炒得沸沸气扬扬的同一件事，却很有一种事不关已的感觉。某某考生晕倒、某某作文零分这类消息只是像一些再普通不过的新闻从眼前飘过。唯一能产生点联想的是又会有多少新生加入到我们的学院，想到又要面对一批执拗的小P孩，吐出一声叹息……改选他们思想的工作，还是交给现实吧。

###六月是毕业的月份。###

最直接的印象是看到一批一批在机械楼前拍学位照的毕业生，宿舍与综合楼间一路毕业甩卖的地摊，还有不知从哪钻出来6毛钱/斤收购旧书的贩子。每年就是这样一批一批人消失，我们也一年年资格变老，然后就是我们也即将消失。那种离别的痛，一年一遍，一丝一丝侵入心中。与已经毕业的学长交流时，无一不告诉我想工作很累，很想回学校，于是知道了四年短暂、知道了就业困难、知道了生存不易，于是不再是小P孩。

去年暑假实习时带我的程老师（学高为师）突然联系我。他两次跳槽，现在又被以前公司挖回去了，老板让他负责开发人员的招聘。他以为我今年毕业，问我想不想去这家公司。听说我明年才毕业，声音有些失望，我却觉得很欣慰。还有一年的时间来学习，明年的这个时候我一定更有资格踏入这烈火般的职场。

###六月是足球的胜宴。###

能看到世界杯的六月，是四年才有一次的。虽然是个伪球迷，但哪个男人不崇拜活力？不像有些同学一样对球星如数加珍，但射门同样能给我带来感观上不逊于他们的享受。2支世界顶尖的队伍，22个强健的男人，1只跳动的皮球，就是有这样的魔力，吸引着数亿观众为之呐喊。足球运动是体能与智慧化合极致，于是宿舍区里骤起的喝彩与叹息也成了一道听觉的风景。

###X是六月里的一堆烦琐。###

冬天都未曾感冒的我竟然在一个闷热的夏天中午起床时发现嗓子剧疼且涕流不止，漱起的时候还咳出了一堆带血丝的痰，当天就用掉了一卷纸。猛吃了两天的药，虽然好了很多，但到了一定的程度就不再好转了。感冒让人没有丝毫胃口，于是吃了几天的方便面。今天一觉醒来又发现左边臼齿疼得厉害，牙龈也肿了起来，让我哼哼了一天

因为感冒让人没状态静心工作，接下的流量统计系统拖了好久才开工，昨天几乎一天的时间老花在了IP数据库的减肥上。半夜12点多才用写出来个IP库减肥程序，26万条记录，本以为跑两个小时就能完成，结果跑到凌晨五点都还没看到要结束的迹象，终于坚持不住，让电脑开着，人去睡觉了。因为记挂着这事，上午10点半就醒了，爬下床终于看到程序显示着转换完成，记录变成了3万多，然后又是工作到晚饭时间。现在的我又病又疼又累又困，可偏偏不想睡觉，唉~

暑假在家里写写程序应该会舒服很多吧，可一想到要把这么沉的电脑运回家就头大，想要本本的念头再次升级……

百步笑五十

发表于 2006-06-03 | 更新于 2018-05-23 | 分类于 扯淡

今日颇觉得电脑上的讯雷5不爽，登录平常喜爱的霏凡软件站寻最新版本。很快在本月下载版第三的位置找到了迅雷(Thunder) v5.1.6.198正式版，觉得它如此争气，果不孚我厚爱。点击后看到的东西却让我非常迷惑，然后大吃一惊，红色加粗文字如下：

1、迅雷下载器是以盗链接(P2SP)的方式来进行文件下载加速的，大量盗用提供本地下载服务的网站链接，增加被盗网站的成本，造成了这些提供本地下载服务网站的直接经济损失。就好象盗打电话，从别人家的电话线上开个口子，接台电话，供别人盗打，而电话费就由被盗打的人付。

2、任何用户只要安装了迅雷下载工具，使用迅雷下载的时候，用户的下载行为会完全被迅雷记录并上传，这样的行为也侵犯了用户的合法隐私，造成了网络安全的隐患。

3、迅雷下载技术跳过了网页环节，使得用户在下载文件的时候，有可能下载到非法网站的内容，使用户可能感染电脑病毒和木马，并对信产部网站监管工作造成不利影响。

4、我们强烈要求迅雷对我们下载站停止盗链行为，但迅雷一直采用回避拖延的态度，因此我们只能暂停迅雷软件的下载，并要求迅雷立即停止对我们的网站盗链接的侵害行为，我们同时保留进一步采取法律措施的权利。

5、我们推荐象FlashGet这类通过网页建立下载，以断点续传为主要功能的免费软件，为用户和网站都带来方便。

一个国内最知名的盗版软件站点倒戈指向国内最知名的下载软件，打的旗号竟然是反盗版。中国有成语五十笑百步，在这只能说是百步笑五十了。不解的是明明没有提供下载链接，这软件怎么能爬上月排行第三的位子。平时用惯了免费破解的用户一大堆，态度在那近400条留言中自见。霏凡这次自取其辱，怕是要失去大片人心。废话不多说，还是去当下软件园看看好了。

来也匆匆去也匆匆，期待已久的软考就像列火车晃过了5月20，而我更像是一个站在道口看着它的路人，毕竟只为它准备了4个晚上*2小时。成绩不上不下，过了固然高兴，没过也只是可惜了那110块报名费。
做完下午题便被鸡翅膀拉到师大后书淘旧书去了。逛逛书店，我与书的故事（此故为过去的意思）浮上心头……
小时候的我除了有多数男孩的调皮捣蛋外，还有他们所没有的安静。头一天还能在外面玩得惊天动地地闯祸，第二天又能一个人在家抱本书呆一天不出门。似乎是很矛盾的两种性格，在那时的我也没觉得什么不正常，现在想想，矛盾是我从来不缺的东西。
爸妈都是工人，家里的书不多，也就没得我挑了，只要是带字的纸我都感兴趣，一些专给我买的童话自然是翻了N遍，还有那些《生活百科知识》、《电工手册》也是拈来就啃，实在没书看的话抱本《现代汉语词典》也能看得津津有味。当时甚至有计划从A到Z把这本字典全读完，最终还是没能实现，毕竟大多数书籍比它有意思得多。看到不懂的就问我爸，他总能给我一个答案，实在是讲不清楚他就说等你到初中学了就明白了。然后我就作仰慕状，觉得老爸真厉害，什么都知道，大概每个儿童都是这样看他们父亲的吧。唯一不明白的是我问他我从我妈哪里生出来的，他有时说是腋窝下，有时说我是路上捡的，有时说是树上摘的，让我纳闷了有十年。
那时我是彻底地迷上阅读了，上厕所都要拿本书进去，一蹲就时半个小时，每天睡觉前都要靠在床上看上两个钟头，近视也由此而生，每过一个学期变深50度，持续了十几年才稳在了750这档。那时最爱的味道是新书里的书香，深吸一口能让我满足无比。对书籍的痴迷让我逐渐产生了一种恋书情结，我从来不折书，怕弄旧了它。也不在书上写字，怕破坏了它的格式，到后来干脆连自己的名字也不写了，在街上让人刻了个章，买来了新书就蘸点印油一按，我的名字就落上去了，就为它比我手写的更整齐，更配得上那些标准的方块字。我的书极少借人，因为以前借别人的书还回来后常常面目全非，甚至直接失踪，后来索性不借了。遇上super好友，实在不好推托，我也会婆婆妈妈或请求或威逼交待一阵，让友人好好爱护，就像要把姑娘嫁到他人家去的母亲一样，显得很神经质。后来由书及物，变得对很多东西都有了这种情节，让我自己都觉得自己太小家子气了，真TM不像个男人。
受我爸影响，我阅读的兴趣主要在科学技术知识上，那时候的梦想可是要当一个爱迪生一样的发明家。最爱的也是对我影响最大的几本书是借同学的几本《十万个为什么》，只有物理、化学、天文卷。那时候的读书是与我的实验分不开的，在书上看到什么有意思的东西就在家仿照实现，为这爱好没少闯祸。自豪的是我爸说的那些我到了初中就能明白的东西经过我的摸索，在小学时就掌握得差不多了，后来我的物理一直是班上前盾。
高中是在市中心读的，走出了那个离家只有200米不到的小校园，天地一下宽广了很多。受同学的影响，开始接触武侠小说，小说的来源主要来自学校附近的租书铺。那时候家里给的钱很少，押金都交不起，竟然把团员证押上了(-_-!!!)。捧着这些不知被多少脏手摸过而乌漆麽黑的小说，游离于快意恩仇的武侠世界，不知做过多少英雄美梦。最不爽的是正看到精彩处突然发现被撕去了几页，推测是哪个曾经的读者大便没带纸，用英雄的故事擦了PP后扔进厕所冲进了化粪池，心中诅咒此人千百遍千百遍……最早看的是李凉，觉得他写的武侠很诙谐，后来看了金庸小说后就心中无它，大有“除却金庸不是侠”的味道，彻底成了他的fans。因为除了金庸以外别的武侠我看不上眼，所以只好把他的小说看了一遍又一遍，至此不彼。上大学后还去买了套四本装的“精装”的《金庸全集》，之所以打引号是因为这是只花了￥35买来的盗版书，以后有钱了一定买套正版 。
除武侠外，高中还读了很多中外名著小说，除了王小波的《时代三步曲》以外都觉得没看金庸过瘾。那时因为常常流连于游戏机厅和网吧，然后毫不奇怪地迷上了计算机。买了一堆计算机书籍天天啃，坚持不懈地看电脑报，最终也选择了这个专业，现在看来那些书都是些菜鸟级的内容。除此以外还买了很多军事武务方面的书籍。
上大学后基本没有读书了，就算读也不是传统意义上的读书，只是抱个手机看了几千万字的各种网络小说而已，纯粹的消遣，让我觉得自己很堕落，曾经那个对书籍如饥似渴的少年哪里去了？于是乎今天逛书市时感觉特别复杂，这两天要务在身也要写下这段故事。
列举一些对我有影响的书或文章：
《十万个为什么》
——带着我深入了科学的神秘世界
《智力名题100例》等
——这是我爸给我买的，加了个等字是因为还有两本我记不起名字了，但里面的题目都很有名，使我解决问题的思路开阔了很多。
《统筹方法》
——这是语文书上的一篇课文，华罗庚写的（我也就是因此才记住了他的名字），教会了我如何最有效率的利用时间，成就了今天我一身的彻底的实用主义。
《金庸全集》
——“飞雪连天射白鹿，笑书神侠倚碧鸳”，绝不仅仅是武侠，金庸小说里的知识非常广泛，让我第一次认识到真正文学家的功夫。他的小说对我的写作风格甚至价值观都有很大的影响。
《青铜时代》《白银时代》《黄金时代》
——王小波的时代三步曲，深刻的黑色幽默，让人笑里含着酸泪，是诙谐与严肃的无缝结合，可惜天嫉英才。
《小霸王电脑学习机说明书》
——对，没错，就是它。这是我计算机与编程的启蒙书，第一次打字，第一次编程全是从它开始的，怀念上面的F - BASIC。

小喇叭开始广播啦

发表于 2006-05-14 | 更新于 2018-05-23 | 分类于 技术

混混噩噩地过了五一，混混噩噩的又过了五一加七，实在是混混噩噩得要抓狂了。电脑开着，心思却丝毫没在屏幕上，总得干点什么吧？于是瞅到了这个白白的盒子……

三个月前在淘宝买了个音频无线转发器，东西寄过来才发现是音频输入是用的音响上那种AV接口，而不是想象中的3.5耳机接口，拖到前两天才托同学帮买了一根转接线。买来电池，接上电脑，打开收音机，果然能接收到电脑的声音输出，只可惜不是立体声的。不爽的就是这玩意得装三颗5号电池，估计也用不了多久，于是心生一念，想把它改成USB供电，并去掉转换线的插头，将线路直接引入。

借来了插入班同学的电烙铁和焊锡，找来根USB转MiniUSB的转换线（MP3上用的，拆了我还真心疼），万用表启子镊子刀子等工具备齐。切掉USB线的MiniUSB端，分出5组线头，分别是最外层的屏蔽层和中间的四芯。用万用表测中间四根芯线可以得到5V和4.6V两种供电，最后选择了5V供电的红黑线，功率应该可以大些。拆开切断音频转接线，发现这线质量还真差，分出四根线头。剩下就是焊接的事情了，这是从小学就玩起的，轻车熟路三两下搞定，完成拼装。

最后的效果很满意，只要电脑开着它就在工作，不用担心电池什么时候用完，信号基本能清晰覆盖周围的寝室。没有去除原有的任何功能，以后也可以方便地用于音响设备或没有USB供电的设备上。

“小喇叭开始广播啦…………”

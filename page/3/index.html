新的开始

发表于 2011-10-27 | 更新于 2018-05-24 | 分类于 职业

新公司是多玩，在科韵路软件园，离家里3公里多一点，上下班骑自行车就可以了。离职后只休息了一天，上午睡懒觉，下午租了个车在软件园转了几圈，确认了公司的方位与路线，免得以后耽误时间。相比之前，每天要早起半小时，但比起挤公交的上班族，依然要幸福太多。广州的公交自行车上线一年多，现在才真正享受到这一种福利，房子买在华景是正确的，老婆是对的。

12号报到，签合同读手册见主管，发了套新台式机，i5+4g内存+500g硬盘，一下子觉得美好了起来。之前准备充分，带了两个加起来24G的U盘，刷刷把环境部署好，发现公司网络带宽达到100M，美好+1。登录YY加入公司的大群，然后很快就加入了羽毛球、电影、摄影3个组织，美好再+1。

别人问我去多玩是不是做管理了，我很忐忑地回答不是，依然做开发，而且还从unix转向了windows，做YY语音的客户端开发。虽然我没正经开发过windows软件，但学习新东西总是能让我兴奋，何况我一直觉得用Eclipse和VisualStudio的程序员都是最幸福的程序员。也要感谢多玩给的信任，我以为员工的优秀在于性格与思考方法，相信我在这里很快会有用武之地。

20、21号做了两天的新员工拓展培训，这种事情我已经是第四次。回忆起第一次参加的情景，少了一些激情热血，多了一些成熟理智。曾花了两年的时间换了职业，又花了两年多的时间换了行业，虽然现在已比一些同龄的杰出者已差了十万八千里，但每次视角的变化都让我看到了更多的东西，人仍然在进步。再次感谢那些帮助我、欣赏我的同事与领导，感谢那些牛逼的人让我知道该做什么，也感谢那些傻逼的人让我知道不该做什么。

到今天，正好工作了两个星期，已经比较适应了新的环境与节奏，能很明显感觉这次不仅是换了个环境，生活方式也往好的方向进了一步。更严格一点的考勤制度让我作息开始规律，食堂让我饮食变得规律，每天40分钟的自行车也比走路有趣很多，心情变得更愉快，又会反过来促进自己做更多有意义的事情，比如长跑、爬山，这样挺好！

继续加油，积极乐观，每一次新的开始，都应是更美好的开始。

有幸与你并肩，不得不说再见

发表于 2011-10-11 | 更新于 2018-05-24 | 分类于 工作

下午六点，我背起背包，跟你们一一招呼：“我下班了，拜拜！”。

你们站起来，眼神复杂……我没再多说什么，因为嗓子不舒服。扭头，眼睛也不舒服。

出门的刹那，人突然一蒙，好像魂魄留在了身后。

就这样，我离职了……
不知道是不是有天意，三年前的今天，我辞职离开亚信南昌，在华为干了半年又重新进了亚信广州，但今天一走，再难回归。

离职的目的是为了换一个行业。从毕业到现在，从亚信到华为，一直在电信业内。这4年里，除了硬件升级为3g网络，中国的电信业务没有什么大的变化。我们做的移动CRM/BOSS系统是一个千万行代码规模的怪兽，每年集团公司都在出新的规范，逼着我们不停的演进，给怪兽换上不同的造型。所有的省公司都绞尽了脑汁，想出各种怪异的促销，却永远是围绕着彩铃、流量包、X信通几个增值业务。我们说实现不了，他们说一定要实现；我们说一个月来不及，他们说一定要来得及。我们就这样被蹂躏者，用各种蹩脚的设计将不可能变成可能，用出差、加班将来不及变成来得及。然后这些摇摇晃晃的功能可能很快被遗忘，或者再这个基础上实现更复杂的业务、甚至被推倒重来，日复一日，疲惫不堪。

以我看，基于人的企业很自然地就有人的属性，比如遗传（或者是学习、模仿）。中国的大型国企事业单位，都遗传了一些国家政府的特征，移动这样中央统筹、地方自治的企业更像。集团的规划很难在各省正确地落地，上下充斥着自大、忽悠、造假，以本位思想驱动追逐增长，没有为用户真正的需求着想。

坐拥如此巨大的资源，当初的中国互联网还是靠它的短信支付才得以拯救，却如此的保守与自大，实在让人失望。拿出一些魄力与资金，招揽国内最优秀的技术人员与产品经理，早一点做即时通讯，腾讯哪有今天的滋润。握着机密的基站数据，在还没有GPS手机的时代精研LBS，现在也该是何等风光。至于微信米聊什么的，没人比移动来做更合适。

垄断的优势还在，纠正思想，招聘能人，电信业的机会仍然无限，只是我等不及了，趁着还算年轻还有人要，果断投身互联网。虽然那边风大浪大，但起码有希望。能为这个时代画上一个像素，也算我的一点小小理想。

亚信这两年离职率很高，我听了各种抱怨，郁闷时自己也会说两句，但我会想得更多一点。曾看到有数据说创业公司有70%都倒闭掉了，亚信走了十几年，上万人的规模，国内行业第一，必然是有其过人之处。时间一旦一久，人很容易忽略环境中的优点，而对缺陷念念不忘。抱怨可以短暂释放压力，副作用是导致怨念的加强，特别是聚众抱怨，消极的思想很快就充斥大脑。我第一次离职时就这样单纯，觉得我们的产品太烂，做的好累，到华为后才知道原来我们这个行业大家不是在比好，而是比不烂，亚信的产品其实有很多优点，公司制度上也比华为宽松很多，所以后来找机会又跳了回来，心态成熟了很多。

即使行业、公司、部门差劲，也不是自己不进取的理由，与其抱怨、逃避，不如从工作中学点什么提升自己，为下一次机会做准备。

所有的公司都是由优秀者驱动，平凡者跟随而前进的。我还未毕业就进了亚信，有幸能与那些优秀者一起，将无数需求转化为了现实，不可能变为了可能。在这里我学到了技术，知道了还有比技术更重要的职业素养，感受到了团队的能量，看到了如何精彩地生活与工作，感染了乐观。

我感谢大家，能和你们并肩是我的无上的荣幸，只是我要换一条自己的路，不得不说再见了，与诸位共勉！

Foxmail 7.0 试用体验

发表于 2011-09-04 | 更新于 2018-05-24

2000年我还在读高中，在某个小网吧里初次接触Foxmail，用它的远程邮件管理帮同学删邮件炸弹。那个时候它还有着浓重的个人软件风格，Logo旁写着作者张小龙。工作后每天面对无数邮件，我尝遍了OutLook、DreamMail、TheBat、ThunderBird…之后，还是留下了Foxmail，因为它的简单轻巧。

我习惯不删工作邮件，为了让邮件更有条理，创建了很多过滤器与目录。而老Foxmail中每一个邮件目录对应磁盘上的一个物理文件，要让文件不太大影响效率，还得按时间建目录进行归档，归档完了还压缩，控制每个存储文件在100M以下，很快就堆了几G。那时候WEB开始流行标签化，而Foxmail虽然能给邮件打标签，但不支持标签管理，比较遗憾。Foxmail6.5提高了搜索速度，加入了搜索文件夹方便邮件组织，但受限于文件存储方式，效果提升有限。

时隔两年，腾讯最近终于吐出了Foxmail7.0，马上下载体验，有不少惊喜。

7.0这次最大的改进在于彻底修改了邮件的存储方式，由之前每个邮件夹对应一个文件，变成每封邮件对应一个文件。存储粒度的减小让邮件摆脱了文件的限制，能支持更加灵活的组织方式，我久盼的标签化也就顺理成章。通过标签化管理，对邮件的存放组织更加符合真实应用，看着舒心用着顺心。

还有邮件在不同目录之间的移动，只须更改文件索引的位置，不用再做数据的分离与追加，可以瞬间完成（类似文件的同盘剪切-粘贴）。更不用像历史版本一样，还要对原目录压缩以收回空间。

另外，我电脑之前速度挺快，后来越来越卡，经常硬盘灯狂闪，换硬盘重装系统都无效，换上了Foxmail7后这几天却表现极好，这让我很怀疑Foxmail老版本在邮件堆积后会影响严重系统IO。

除了邮件存储，7.0开始全面支持MicrosoftExchange邮件服务，这个我用不上，估计是为了配合QQmail最近的终端Exchange特性。另外最大的改进是更强大的邮件编辑器，老版这点之前经常我抓狂，改个颜色或字号老不一致，这次总算满意了。搜索的速度提升不少，新增的附件预览和鼠标手势（strokeit用户对此无视）比较贴心。

仅标签化这一改进已经有了让我不得不升级的理由，但作为一个软件控+细节控，我还是要列列bug，提提建议。

bug：

  • 草稿箱的邮件，按回车键打开是阅读模式，且邮件无法编辑（双击打开编辑正常）
  • 转发邮件，将光标移到原邮件内容中，按Esc键，会插入自己的邮箱账号名
  • 软件帮助菜单，链接到网址http://help.foxmail.com/foxmail7/help.html#，竟然显示“正在建设中…”，然后是一片空白
  • 创建过滤器，输入的搜索条件含有空格，如“x y”，保存会发现被保存了“x” And “y”，意义完全不一样了
  • 机器休眠一夜，或者过了0点，打开界面收取到邮件，新邮件会被放入昨天的列表，重新切换邮件夹后正常（这个bug很多年了）
  • 在我的两台机器上，Office文档预览不成功
  • 对在“存档文件夹”中的邮件右键 - 复制/备份 - 存档，会在存档文件夹下再创建一个存档文件夹的子目录，有点无聊
  • 没有新邮件时，点击收取新邮件，弹出的“正在收取”对话框会显示上一封邮件的标题，看着不舒服
  • 邮件列表视图中的发件人会根据通讯录进行翻译，但附件列表视图中发件人不会

还有很多小地方，一时数罗不出，以后补充。

建议：

  • 搜索文件夹支持搜索条件“最近xx天”，而不是仅基于一个固定的日期之前或之后
  • 过滤器还可以更加丰富，要方便测试，再不济也要将老版本中“使用过滤器处理选中邮件”的功能弄回来
  • 继续强化标签列表，支持标签的分层与隐藏
  • 附件预览支持自定义格式或“以文本方式预览”，很多附件都是后缀名怪异，其实都是文本的。
  • 建议增加过滤器、反垃圾规则的云端保存
  • 能在邮件列表中直接输入，按拼音首字母查找发件人姓名或主题，类似Total Commander的快速搜索，7.0的搜索界面设计简直是种倒退
  • 既然能用贝叶斯过滤垃圾邮件，那自然能用来做邮件归类，手工创建过滤器什么的太out了
  • 既然跟着QQMail后已经增加了RSS和日程管理，那也可以考虑实现记事的功能了，对一个工作用的软件来说，记事比RSS更有意义，当然这些都得存在QQMail云端

新版本在界面上的修改很大，有极多的细节可以改进，如用词不当，还有控件位置导致的视线、鼠标移动过长等等，我再列下去就太龟毛了，但请体谅下下一个老用户的憋闷太久终于释放的心情。

总的来说QQMaill团队花了两年时间，让Foxmail7连个Beta的帽子都不戴就出门，还是有点不负责任。最初发布的版本，有6.5导入的账号不能收取邮件的问题，让我受挫不小，好在上周发布的Build79版本解决了这一问题。但瑕不掩瑜，试用几天下来，惊喜程度还是超过了缺陷带来的影响，推荐追求速度的朋友使用。对那些不怎么在意6.5效率的朋友，还可以等几个月。

收件箱是个信息的大泥塘，无论是Web端还是客户端，让用户能更及时更有效的分拣与处理信息，都应当是最核心的需求。QQMail与Foxmail这几年增加了很多功能，追求新增长的同时还是要不忘根本，在这方面永远还有创新可挖。

贷款计算器Excel版

发表于 2011-07-14 | 更新于 2018-05-24 | 分类于 项目

昨天兴冲冲到农行申请部分提前还款，被银行大姐鄙视了一把。

对话1：

她：你提前还6w5其实意义不大

我：我之前在网上查过，好像能省不止6w5的利息

她：不可能有那么多的

我：那麻烦你帮我算算，我假如今天提前换6w，到最后能少交多少利息？

她一脸不耐烦算了几分钟，最后说：7w多

我：能省7w多，这对我意义太大了

她：。。。

（大姐是你不懂呢还是真觉得7w块没意义啊？）
对话2：
我：表格你帮我看看，我提前还贷6w5，想缩短期限，这个调整期数一栏我该怎么填？

她：你怎么想的嘛？想怎么填就怎么填！

我：我听说有很多种，比如月供不变缩短期限，或者期限不变减少月供的。我就是不知道具体怎么算，咨询你嘛！

她：那要看你要怎么样，你怎么填利息就怎么算

（费了半天劲还是没说不清）

我：那你帮我算算要是减两年的话，我调整后月供多少？

她：xxxx

我：那就行了！
吃了没文化的亏，回来后在网上找了不少提前还贷计算器想仔细算一下，能搜到的web版计算器基本都是一个模样的，不支持浮动利率，不支持多次提前还贷，甚至全都有一个可笑的bug：输入提前还贷0元，最后都计算出能省几千大洋。

好，不话唠了。最后我学习了一下等额本息与等额本金的计算方式，用Excel写了一个按揭与提前还贷计算器，基本合意。

特性如下：

  1. 支持等额本金与等额本息两种计算方式
  2. 支持公积金与商业贷款两种利率，支持利率折扣
  3. 支持调息后录入新利率，支持浮动利率（首年按贷款时间点的利率，之后按每年当年1月1号的利息值）
  4. 支持计算每月还款额度，本金与利息的组成，本金剩余与已缴纳利息总计
  5. 支持多次提前还贷，自动计算每阶段的月供、利息
  6. 支持打印（废话了，excel还能不支持打印？）

喜糖记

发表于 2011-06-22 | 更新于 2018-05-26 | 分类于 生活

终于完婚了，没有婚纱，没有仪式，只是五一与端午分别在两人的老家宴请了一下亲朋。都是两边父母在辛苦张罗，我们两个当事人只算是“参加”了一下。庆幸遇到了一个和我一样的她，不在乎那些繁文缛节，一心只想过和别人不一样的生活。

在我们生活和工作的根据地广州，我们办不起隆重的婚礼，但给朋友们准备了心意满满的喜糖，谢谢每一个你与我们的相伴！

喜糖盒是老婆从淘宝众多款式中综合外观、大小、成本挑出来的，高价买回了两个样品后确定购买。寄来的是N张底壳、N张上盖、N张印有我们俩英文名与Q版造型的吊牌、N个蝴蝶结、N朵小纸花，每一个盒子需要自己手工折叠与粘贴，成形后是个5cm边长的小方盒。

糖果也是在淘宝上找了个大卖家，先买了一斤样品糖试尝，从中选定了10种最好吃的确认购买，不过一次吃掉一斤糖可不仅仅是件甜蜜的事:)。

折盒子是个枯燥漫长的阶段，大概20秒能折一个底或顶盖，还要半分钟的时间用热熔胶粘吊、牌蝴蝶结与纸花。我们俩折腾了一星期才弄完，那几天整个客厅都是材料与半成品。

装糖也不是件简单的事，每个盒子装哪些糖装多少颗都得规划好。

高效能Windows人士的N个习惯之一：启动篇

发表于 2011-04-26 | 更新于 2018-05-24 | 分类于 技术

接触电脑十多年，经历了各种折腾阶段，这几年开始沉静下来，不再追求花哨的界面与应用，只注重工作的效率，逐渐养成了一套自己的操作习惯，感觉不错，特撰文分享。标题借用了一下《高效能人士的七个习惯》，这些技巧大多提炼自善用佳软与小众软件两站，特别感谢那些作者非凡的分享。

此文主要结合我自己的应用，分别讨论启动、查找、鼠标、剪贴板管理、快捷键、文件管理、文笔编辑、命令行这几个Windows中最常用，也是最繁琐、最乏味的操作，看看如何结合特殊的软件与习惯，将它们变得高效而生动，并且每一篇后面都提供一些我的补充心得。

启动篇

我们每天打开电脑启动Windows，主要是在和里面的应用软件打交道。大多数人系统中可能安装有几十个软件，常用的不过十几个，每天在程序列表的高山与桌面图标的海洋中苦苦搜寻那几个名称早已倒背如流的常用软件，是否觉得辛苦？我曾试过很多种桌面布局或图标分类软件，也曾创建过四个边框都是密密麻麻图标的工具栏，发现都走不出困境。所有基于图标的工具或方案，都要求我的人脑做出一个“软件名称 -> 软件图标 -> 图标位置”的思考与选择，最后用鼠标点击启动。这些工具与方案，只能让这个思考与选择的过程变得更容易。但是，我们真的需要图标与鼠标吗？越来越丰富的界面让很多人都忽略活着忘记了其实DOS命令行就是一种很高效的启动方式，虽然是在图形操作系统中，我们仍然可以放弃图标或菜单的包装，直接用名称来进行启动软件。

不需要专门的工具，Windows原生就支持此功能：把最常用软件的快捷方式都扔到一个文件夹中，将此目录的路径追加到环境变量PATH的后面。要启动哪个软件，按Win + R打开快速运行栏，输入快捷方式名称回车就行。为了方便记忆，软件快捷方式名称尽量改成短小的英文简写，Internet Explorer就叫IE、Firefox就叫FF、植物大战僵尸就叫PVZ……对于名称较长的快捷方式，下一次启动时输入两个字母后就能用键盘上下选择再回车。相比图标启动方式，大脑只需要记忆，不须要联想与图标检索，再加上省略了鼠标点击，双手可不离键盘，效率高出了数倍。

一段时间的习惯后，桌面上的快捷方式都可以删除掉，腾出的空间可以像我一样当临时目录用。

补充心得

资料数据不存C盘，这是Windows多年不变的真理。可将“我的文档”重定向到“D:\MyDocs”目录，将桌面重定向到“D:\Mydocs\Desktop”目录，而专门放快捷方式的目录就放桌面上。新装完什么软件后，把生成的桌面图标改个易记的名字拖进此目录即可。尽可能用绿色软件，那些重装系统后必须再次安装的软件也保持在固定的位置，这样快捷方式目录中的内容以后基本不用再维护。

香港一日行

发表于 2011-04-17 | 更新于 2018-05-26 | 分类于 生活

上周日去了趟香港，应老婆大人之命写一篇游记，不低于800字。

先说说通行证，中国是世界人口最多的国家，GDP全球排名第二，但持中国护照可以免签通行的国家只有38个，全球排名88，次于韩国、印度、蒙古、越南、老挝等等邻居，也就比朝鲜还强一点。更郁闷的是，虽然香港14年前就回归中国了，大陆国人去香港也仍然得办理签证。目前国内各个城市一般都开通了办理港澳通行证的窗口，首次申请15个工作日内可完成。通行证分两种，一种是个人旅游签证（G签），能自由出入香港；一种是团队旅游签证（L签），必须随团进入香港，可自由离开。

搞定了通行证再准备交通，一种是买省港通巴士票，来回100多点。另一种是找旅行社报名，如果L签那只能选后者。广州的旅行社一般都有香港自由行的线路，我们报了东方国旅，每人98包来回车费，必须按点按车入港离港，我的L签得多交50团签费，两人共RMB246。临行前一天，收到随行导游叶生的短信，再次通知了出行安排。然后提前给手机开通国际直拨与国际漫游、记录要买的东西、查攻略、计划游逛路线、下载好离线地图（手机漫游上网资费比较贵），折腾到2点才睡。

周日，辛苦的一天开始了。5点半起床洗漱，下楼取了三千块钱（后来发现其实不是很有必要），打车到冼村路口集合，在M记吃个早餐。6：45，登车出发。导游叶生人很不错，分别用普通话与粤语介绍了注意事项，耐心回答了大家很多问题。据说香港的法律很严格，扔垃圾、闯红灯、在交通工具内吃东西、在室内抽烟，都可能将导致数百至数千的罚款，还得上法庭认罪。由于香港严格禁烟，给香烟征收了极高的税，为避免走私，从大陆入港每人携带香烟数目不能超过19支。香港的黄金、电子产品相对便宜，但带回大陆超过额度的话需要另外交税，未申报但被抽检到须另交罚金甚至按走私论处。还有在香港法X功组织不违法，经常在街头向人派发传单报纸，可以在香港阅读，但千万不要带回国内，否则查到将相当杯具……

上午9点抵达深圳福田关口，持L签的游客随导游一起出关，其他人自由出关，出关后L签和G签就没区别了。两岸关口间有香烟免税店，但除非不打算进香港了，否则还是等回来的时候再买。前行一段，进入香港关口，这就到达资本主义的天下了:)。

关口能兑换港币，但汇率相当坑爹，最近的港币汇率大概是0.84，人民币100元兑换港币应该是119，这只能兑换到110。不急用的话到了市内再换，街头有很多兑换点，比例能到118。在香港出行买一张八达通比较好，公交、地铁、便利店都可以使用，否则带一堆2元5元的硬币是很沉的，关口和地铁站都有售，应该是50港币。

队伍集合与等车耽误了很长时间，11点40才到达第一站黄大仙。一路上山山岭岭，没有想象中那么土地紧张，但这边房子一盖就是几十层，格局也很紧凑。

苏宁维权记

发表于 2011-03-21 | 更新于 2018-05-25 | 分类于 生活

过年期间家里置办了很多腊味，以前170升的冰箱空间不足了，加上服役几年，有些冰霜、漏水的小毛病，下决心换个空间大的。按广东的国家政策，老冰箱还能选择以旧换新折个几百。

2月26日带着老婆老妈跑到国美维多利新活馆踩点，看上了一款西门子KK28F73TI，价格最低5700。回家后第二天（27日）在苏宁易购网站上找到了同款型号，只要5291，立刻下单。填写订单过程中，特别注意了以旧换新相关项，并按网站提示在填写发票抬头时输入的是老婆的名字（因为我本人户口还在老家，老婆的深圳户口才属于广东省，能享受以旧换新，去年买电视办以旧换新的时候就因用了我的名字而重填过一次发票）。选择货到付款，搞定！

28日上午，接到苏宁客服电话，确认送货时间，下午冰箱就到了家。刷卡完成后手机收到一个确认码，送货师傅将此码填在确认单上，再给了我一张发票领用单。当时觉得苏宁服务相当迅速，24小时内就到货，而且付款确认机制也很便捷周到。问及以旧换新事宜，他们说他们不清楚，让我问客服。电话客服被告知须持发票领用单至苏宁正佳广场店换取发票，并办理以旧换新手续。
